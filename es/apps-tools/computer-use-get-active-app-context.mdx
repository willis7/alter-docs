---
title: "Obtener contexto de aplicación activa"
description: "Capture el contenido de la ventana activa y el texto OCR para flujos de trabajo de uso informático"
---

Utilice esta herramienta para leer lo que está actualmente visible en una aplicación o ventana del navegador antes de realizar alguna acción.

## Qué hace

- Recupera contenido de la ventana de la aplicación activa de forma predeterminada.
- Acepta `windowID` opcional (`pid/title`) para apuntar a una ventana específica.
- Devuelve contenido de texto extraído.
- Agrega texto OCR de capturas de pantalla cuando sea útil.

## Usos comunes

- Lea el estado de la página o la aplicación antes de hacer clic.
- Extraer contenido visible para resúmenes.
- Construir un contexto estable antes de la automatización de varios pasos.

## Aporte

- `windowID` (opcional): apunta a una ventana específica de `list_windows`.
