---

Computer Use le brinda a Alter una capa de ejecución de UI para flujos de trabajo de escritorio. Puede inspeccionar ventanas, interactuar con elementos de la interfaz de usuario y controlar flujos de varios pasos.

## Conjunto de herramientas

- [Obtener contexto de aplicación activa](/apps-tools/computer-use-get-active-app-context)
- [Realizar](/apps-tools/computer-use-perform)
- [Espera](/apps-tools/computer-use-wait)
- [Desplazarse](/apps-tools/computer-use-scroll)

## Flujo típico

1. Lea el estado actual de la interfaz de usuario con **Obtener contexto de aplicación activa**.
2. Active acciones de la interfaz de usuario con **Realizar**.
3. Utilice **Desplazar** cuando el elemento de destino esté fuera de la pantalla.
4. Agregue **Espera** entre los pasos para la carga o las transiciones de estado.
