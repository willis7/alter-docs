Exa es un proveedor centrado en búsqueda web para IA.

## Cómo se posiciona Exa

- "La mejor API de búsqueda para IA"
- Foco en calidad y benchmarks
- Endpoints para búsqueda, contenidos, respuesta, rastreo e investigación.

## Cómo conectar

1. Abre **Administrador de Herramientas**.
2. Agregue Exa en **Servidores MCP remotos** o en **Galería de aplicaciones**.
3. Pega tu clave API.
4. Active el conector.
