Después de tus primeras tareas exitosas, esta es la forma más rápida de subir de nivel.

## ¿Qué te ayuda a hacer?

Utilice espacios de trabajo para mantener listo el contexto de proyectos grandes y luego utilice acciones para repetir tareas de alto valor con indicaciones consistentes.

<CardGroup cols={3}>
  <Card title="Workspaces" icon="folder-tree">
    Mantenga listo el contexto del proyecto para que Alter pueda trabajar rápidamente en conjuntos de archivos grandes.
  </Card>
  <Card title="Actions" icon="zap">
    Guarde indicaciones reutilizables para tareas recurrentes y una calidad de salida constante.
  </Card>
  <Card title="Tool scope" icon="sliders-horizontal">
    Habilite solo las herramientas necesarias por flujo de trabajo para una mayor confiabilidad.
  </Card>
</CardGroup>

## Cree su primer flujo de trabajo repetible

<Steps>
  <Step title="Create one workspace">
    Agregue las carpetas y archivos de un único proyecto que toque cada semana.
  </Step>
  <Step title="Create one action">
    Convierta una tarea recurrente en una acción reutilizable (por ejemplo, resumen semanal o notas de la versión).
  </Step>
  <Step title="Constrain tools">
    Habilite solo las herramientas necesarias para esa acción y luego ejecute el flujo de un extremo a otro.
  </Step>
</Steps>

<Tip>
  Comience con un espacio de trabajo y una acción. Las automatizaciones pequeñas y confiables escalan mejor que las configuraciones genéricas grandes.
</Tip>
