Utilice esta página para conectar rápidamente Alter con los archivos y carpetas que son importantes para su tarea.

## Agregar contexto desde Finder o aplicaciones

Puedes agregar contenido arrastrando archivos o carpetas al notch o a The Hub. También puedes usar **Clic derecho > Agregar a Modificar**.

## Contenido admitido

<CardGroup cols={2}>
  <Card title="Documents" icon="file-text">
    `.xlsx`, `.docx`, `.pdf`, `.txt`, `.md`, `.org`, `.rtf`
  </Card>
  <Card title="Images" icon="image">
    `.jpg`, `.png`, `.heic`, `.gif`, `.tiff`, `.svg`
  </Card>
  <Card title="Web content" icon="globe">
    `.html`, URL y texto arrastrado
  </Card>
  <Card title="Email" icon="mail">
    `.eml` y mensajes de Apple Mail
  </Card>
  <Card title="Audio" icon="music">
    `.mp3`, `.wav`, `.m4a`, `.aac`, `.flac`
  </Card>
  <Card title="Code" icon="code">
    Archivos fuente, archivos de configuración y scripts
  </Card>
</CardGroup>

<Note>
  Alter admite caídas de directorio, OCR para imágenes y archivos PDF y detección automática de codificación de texto.
</Note>

## Transcripción de audio y vídeo

Cuando agrega audio o video, Alter inicia la transcripción automáticamente.

<Steps>
  <Step title="Drop one or more files">
    Arrastra archivos multimedia a la muesca o a The Hub.
  </Step>
  <Step title="Wait for processing">
    La transcripción comienza automáticamente.
  </Step>
  <Step title="Ask targeted questions">
    Pruebe: "Resumir decisiones clave" o "Enumerar elementos de acción por propietario".
  </Step>
</Steps>

<Warning>
  Para el análisis de imágenes, utilice un modelo con capacidad de visión.
</Warning>
