Usa esta guía para agregar Alter como provider personalizado en TypingMind.

## Abrir configuración de modelo personalizado

En TypingMind, ve a:

`Settings` -> `Models` -> `+ Add Custom Model`

## 1) Configuración básica

Configura los campos así:

- `Name`: `Alter`
- `API Type`: `OpenAI Chat Completions API`
- `Endpoint URL`: `https://alterhq.com/api/v1/chat/completions`
- `Model ID`: elige uno de los IDs compatibles que aparecen abajo
- `Context Length`: valor predeterminado o el que prefieras

![Configuración básica de TypingMind](/images/integrations/typingmind-basic-config.png)

## 2) Autenticación

En la sección Authentication:

- `Authentication Type`: `No authentication`

Nota: pasaremos la clave API manualmente con Custom Headers en el siguiente paso.

## 3) Configuración avanzada (obligatoria)

Expande `Advanced` al final y agrega un par de Custom Header:

- `Key` (izquierda): `Authorization`
- `Value` (derecha): `Bearer <YOUR_API_KEY>`

Reemplaza `<YOUR_API_KEY>` con tu clave real de Alter API.
Asegúrate de que haya un espacio entre `Bearer` y la clave.

![Header de autorización en TypingMind](/images/integrations/typingmind-auth-header.png)
