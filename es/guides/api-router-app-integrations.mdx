## Checklist de integración

1. Configura la URL base en `https://alterhq.com/api` (o `/v1` si es necesario)
2. Configura el header de autenticación como `Authorization: Bearer YOUR_API_KEY`
3. Usa IDs de modelo en formato `Provider#Model-name`
4. Ejecuta una llamada de listado de modelos antes de la primera solicitud
