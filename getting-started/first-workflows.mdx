## Start with high-value workflows

Pick one and run it end to end:

1. **Writing help:** Rewrite or summarize selected text
2. **Meeting support:** Capture and summarize a call
3. **Task execution:** Ask Alter to perform a tool-based action

## Best practice for new users

Before heavy tool use, enable **Flow** and keep your context clean. This improves speed and reliability.
