### Navegação Global

Disponível através das janelas (a menos que a janela alvo já esteja ativa).

- ** Shift + Cmd + H**: Abrir o Hub Alter
- ** Shift + Cmd + E**: Editor de Ação Aberto
- ** Shift + Cmd + T**: Abrir o Tools Manager
- **Alt + Espaço** (padrão): Open notch — configurável em **Configurações > Predefinições > Abrir o entalhe de teclas**
- **Alter atalho do menu rápido**: Abra Alter de qualquer lugar — configurável em **Configurações > Geral > Acesso rápido**
- ** Tecla de voz para texto**: Iniciar entrada de voz — configurável em **Definições > Ditação > Atalho** (padrão: tecla Fn)

### Painel Rápido

Quando o painel rápido está aberto (a partir do entalhe):

- **←**, **↑**, **→**, **↓**: Navegar entre objetos
- **Enter**: Selecione um objeto

### Secção de Contexto

- **←**, **↑**, **→**, **↓**: Navegar entre contextos
- ** Barra de Espaço**: Adicionar/remover contexto(s) selecionado(s)
- ** Cmd + K**: Remover todos os contextos da conversa ativa
- **Cmd + V**: Colar o conteúdo da área de transferência para adicionar ao contexto

### Caixa de Pedidos

- **Entrar**: Perguntar com seu modelo ativo
- **Cmd + Enter**: Prompt usando pesquisa web (via Gemini web)
- ** Shift + Enter**: quebra de linha
- **@**: Adicionar elementos ao contexto
- **#**: Mostrar histórico de conversas Alter
- **>**: Mostrar o histórico das transcrições das reuniões
- **/**: Lista e escolha o seu modelo
- ****: Mostrar o menu de ajuda

### Janela de Hub Alter

- **Cmd + F**: Pesquisa de foco (setas para navegar resultados)
- ** Cmd + B**: Alternar a visibilidade da barra lateral
- **Cmd + T**: Página nova (iniciar novo chat)
- **Cmd + J**: Criar Workspace (requer contexto ativo)
- **Cmd + Opt + O**: Mudar para o Workspace...
- **Cmd + K**: Limpar todos os contextos (requer contextos ativos)
- **Cmd + Delete**: Excluir conversa (necessita de conversa ativa)
- **Cmd + W**: Fechar a aba (fecha a janela se for a última aba)
- ** Cmd + Opt + W**: Fechar todas as abas (necessários > 1 abas)
- **Cmd + Opt + ←**: Página anterior (requer > 1 aba)
- **Cmd + Opt + →**: Página seguinte (requer > 1 tab)
- **Cmd + 1...9**: Mudar para um índice de tabulação específico

#### Workspace (exige Workspace seleccionado)

Não existem atalhos predefinidos; estes podem ser definidos através das Configurações do Sistema macOS:

- Configuração do Workspace aberto
- Renomear Workspace
- Definir permissões
- Alternar caixa de areia
- Iniciar/Parar o recipiente da caixa de areia
- Abrir terminal na área de areia

### Janela do Editor de Ação

- **Cmd + F**: Focar a pesquisa
- **Cmd + N**: Nova ação
- **Cmd + D**: Duplicar ação (requer ação selecionada)
- **Cmd + E**: Exportar ação (requer ação selecionada)
- **Cmd + O**: Importar a acção
- **Cmd + Delete**: Delete action (requer ação selecionada)
- ** Cmd + W**: Fechar janela
- ** Cmd + [**: Secção anterior
- **Cmd + ]**: Próxima seção

### Janela Tools Manager

- ** Cmd + W**: Fechar janela
- **Cmd + F**: Focar a pesquisa
- **Quando "Skills" é selecionado**: Cmd + O (habilidade de importação), Cmd + R (lista actualizada)
- **Quando "Remote MCP Servers" é selecionado**: Cmd + N (Adicionar servidor), Cmd + R (Refresh list)
- **Quando "Ações Rápidas" é selecionada**: Cmd + N (Nova ação rápida), Excluir (Excluir ação rápida, requer seleção)

### Painéis de Reuniões ao Vivo

Disponível apenas durante as reuniões:

- ** Cmd + Shift + ↓**: Activa/ desactiva as legendas ao vivo
- ** Shift + Cmd + →**: Alterna o bloco de notas ao vivo

### Configuração do Alter

- **Cmd + ,**: Configurações abertas (quando a janela Alter estiver visível)
