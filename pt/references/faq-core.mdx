## Como abro o Alter?

1. Hover sobre o entalhe.
2. Pressione `Alt+Space`.
3. Segure `Fn` e fale.

![Abrir Alter](/images/getting-started/open-notch.gif)

Veja [Começar](/guides/getting-started) para mais detalhes.

## Como faço para adicionar minha própria chave de API?

1. Abrir configurações (`Cmd+,`).
2. Vá para API Keys.
3. Selecione provedor e digite a chave.

![Chaves personalizadas da API](/images/models/custom-api-keys.webp)

Veja [Como: Use sua própria chave API](/how-to/use-byok) para instruções detalhadas.

## Como faço para adicionar contexto a um prompt?

- Arraste e solte arquivos no entalhe.
- Clique com o botão direito e escolha Adicionar ao Alter.
- Use o `@` para adicionar contexto a partir do prompt.

![Drag and drop](/images/context/drag-and-drop.gif)

Ver [A janela Alter](/guides/alter-window#the-context-section) para mais detalhes.

## Como faço para alterar as configurações rapidamente?

- Abra Alter e pressione `Cmd+,`.
- Ou abra o menu de três pontos -> Configurações.

![Menu de ajustes](/images/settings/settings-menu.png)

Ver [Alter Settings](/guides/alter-settings) para mais detalhes.

## Como configuro o ditado?

Configurações -> Ditação, então escolha processador, linguagem e substituições.

![Dictação](/images/settings/dictation.webp)

Veja [Guia de Ditação](/guides/dictation) para instruções completas de configuração.
