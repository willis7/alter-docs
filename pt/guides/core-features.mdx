## Interagindo com seus aplicativos

### AppSense - Compreendendo seu Mac

Ao contrário dos métodos tradicionais de captura de tela, o AppSense lê dados reais diretamente de aplicativos macOS, permitindo que Alter acesse informações além do que é visível em sua tela.

<CardGroup cols={2}>
  <Card title="Integração Profunda" icon="plug">
Acessar dados da aplicação, não apenas imagens
  </Card>
  <Card title="Conscientização do Contexto" icon="brain">
Entenda no que está a trabalhar automaticamente.
  </Card>
</CardGroup>

#### Aplicação Actual

O Alter sugere automaticamente usar a sua janela activa como contexto:

<Steps>
  <Step title="Procure a sugestão">
Clique em ** Aplicação atual** abaixo da caixa de prompt
  </Step>
  <Step title="Ou usar teclado">
Pressione `↓` então `Enter` para selecioná- lo
  </Step>
</Steps>

## Modo Seguir

O Modo Seguir é uma característica central do Alter que permite sincronização em tempo real de dados de aplicativos, documentos e arquivos ativos durante as sessões de chat.

### Como Funciona

<AccordionGroup>
  <Accordion icon="refresh-cw" title="Monitorização em tempo real">
O Modo de Seguir monitora continuamente o estado das aplicações e dos ficheiros com os quais está a trabalhar, proporcionando ao Alter um contexto abrangente sobre o seu fluxo de trabalho actual em cada turno da conversa.
  </Accordion>
  
  <Accordion icon="lightbulb" title="Exemplo prático">
A trabalhar numa landing page? Enquanto você faz alterações e tem Alter como seu treinador de marketing, Alter vê as mudanças que você fez sem precisar explicar o que você fez.
  </Accordion>
</AccordionGroup>

### Usar o Modo Seguir

<Steps>
  <Step title="Adicionar contexto">
Adicionar os aplicativos ou documentos que deseja incluir à lista de contexto
  </Step>
  <Step title="Começar a conversar">
Alter rastreia automaticamente o estado dos aplicativos selecionados
  </Step>
  <Step title="Fazer perguntas">
Perguntar sobre o conteúdo mesmo que não esteja visível atualmente na tela
  </Step>
  <Step title="Continua a trabalhar.">
Interaja com os aplicativos selecionados ou atualize documentos enquanto conversa
  </Step>
</Steps>

<Tip>
** Dica profissional:** O Modo de Seguimento é perfeito para a criação de conteúdo de forma longa. Adicione seu documento ao contexto uma vez e continue escrevendo enquanto Alter oferece feedback em tempo real.
</Tip>

## Workspaces

Workspaces são um recurso poderoso que permite salvar, organizar e atualizar automaticamente coleções de pastas, arquivos e aplicativos para fluxos de trabalho eficientes assistidos por IA.

<CardGroup cols={3}>
  <Card title="Respostas Mais Rápidas" icon="zap">
Recuperar apenas arquivos relevantes para suas consultas
  </Card>
  <Card title="Eficiente de Custo" icon="credit-card">
Reduza os custos de processamento com indexação inteligente
  </Card>
  <Card title="Lida com grandes projetos" icon="folder-open">
Trabalhe com pastas grandes e bases de código de forma eficiente
  </Card>
</CardGroup>

### Como funciona o Workspaces

Workspaces crie um mapa indexado dos seus recursos selecionados. Os arquivos são convertidos automaticamente:

Tipo de arquivo □ Conversão
--------------------------
*PDFs** * Markdown com marcadores de página *
□ ** DOCX** □ Markdown format
* XLSX** * Markdown/CSV
Imagens** Extração de texto OCR
• **Diretórios**

<Note>
Os arquivos de áudio e vídeo ainda não são suportados por Workspaces. Transcreva-os primeiro, caindo em Alter.
</Note>

### Editar os Ficheiros no Workspaces

<Tabs>
  <Tab title="Editar Simples">
Editar ou criar arquivos individuais usando linguagem natural:
    - Modificar o código
    - Atualizar documentação
    - Funções de refator
    - Criar novos ficheiros a partir do zero
  </Tab>
  
  <Tab title="Multi Editar">
Editar vários arquivos simultaneamente:
    - Perfeito para refatoração
    - Atualizar as importações entre arquivos
    - Alterações consistentes na base de códigos
  </Tab>
  
  <Tab title="Bash Tool">
Executar comandos bash para operações avançadas:
    - Executar scripts
    - Operações em lotes
    - Automação complexa do fluxo de trabalho
  </Tab>
</Tabs>

<Info>
**Primeira Segurança:** Todas as operações de edição são sandboxed usando políticas Seatbelt com verificações de segurança para execução segura.
</Info>

### Criando um Workspace

<Steps>
  <Step title="Abrir o Hub">
Pressione `Shift + Cmd + H` ou selecione na barra de menu
  </Step>
  <Step title="Navegar até Workspaces">
Encontrar a secção Workspaces na barra lateral
  </Step>
  <Step title="Criar novo">
Clique no botão **+** no selecionador de Workspace
  </Step>
  <Step title="Adicionar arquivos">
Use o botão "Adicionar arquivos" para selecionar pastas e arquivos
  </Step>
</Steps>

### Armazenamento Workspace

Workspaces são armazenados localmente em:
```
~/Library/Application Support/Alter/Workspaces/
```

Cada Workspace contém:
- Definição de Workspace (`.alterworkspace`)
- Seus arquivos e pastas
- Arquivos convertidos em cache para acesso rápido

## Alter Acções - Fluxos de trabalho

Alter Ações são prompts avançados configurados com gatilhos e instruções.

### Criando ações

<CardGroup cols={2}>
  <Card title="Editor de Ação" icon="edit">
Acessível na barra de menus para controle total
  </Card>
  <Card title="Criar rapidamente" icon="plus">
Digite "Criar > Alter Action" na caixa de avisos
  </Card>
</CardGroup>

<Note>
As ações incorporadas não podem ser editadas, mas podem ser duplicadas e personalizadas.
</Note>

### Seleção do modelo sob medida

Atribuir modelos de IA específicos a ações individuais para um controle preciso.

#### Por que usar modelos diferentes?

<AccordionGroup>
  <Accordion icon="shield" title="Primeiros fluxos de trabalho de privacidade">
Usar modelos locais para tarefas sensíveis enquanto mantém modelos de nuvem para consultas gerais
  </Accordion>
  
  <Accordion icon="gauge" title="Desempenho Especializado">
Atribuir modelos rápidos e otimizados como Qwen3 para automação ao usar modelos criativos para escrever
  </Accordion>
  
  <Accordion icon="piggy-bank" title="Otimização de custos">
Use modelos locais leves para tarefas simples e reserve modelos poderosos para operações complexas
  </Accordion>
</AccordionGroup>

#### Configuração

<Steps>
  <Step title="Abrir o Editor de Ação">
Clique nos três pontos na janela Alter
  </Step>
  <Step title="Selecionar ação">
Escolha a ação Alter para personalizar
  </Step>
  <Step title="Ir para Avançado">
Encontre a seção "Avançada"
  </Step>
  <Step title="Escolher modelo">
Selecione o modelo específico para esta ação
  </Step>
  <Step title="Gravar">
A ação agora usa seu modelo escolhido
  </Step>
</Steps>

### Personalização do nível de ação Tool

Escolha exatamente quais Tools cada ação pode usar.

#### Benefícios

<CardGroup cols={2}>
  <Card title="Execução Mais Rápida" icon="zap">
Limitar as Tools apenas ao que é necessário
  </Card>
  <Card title="Melhor Precisão" icon="target">
Impedir que IA escolha Tools irrelevantes
  </Card>
  <Card title="Contexto- conhecimento" icon="brain">
Criar fluxos de trabalho especializados por ação
  </Card>
  <Card title="Eficiente de Custo" icon="credit-card">
Reduzir o uso do token
  </Card>
</CardGroup>

#### Configuração

<Steps>
  <Step title="Abrir o Editor de Ação">
Clique em três pontos → " Editor de ação"
  </Step>
  <Step title="Selecionar ação">
Escolha a ação Alter
  </Step>
  <Step title="Procurar a secção Tools">
Veja em configurações "Avançadas"
  </Step>
  <Step title="Escolher Tools">
Selecione Tools específicas ou desabilite todas
  </Step>
</Steps>

#### Tool definir estratégias

- ** Mínima:** Uma ou duas Tools para ações focadas
- **Grupo:** Tools relacionadas para fluxos de trabalho específicos
- ** Acesso completo:** Todos os instrumentos para acções de carácter geral
- ** Nenhum Tools:** Para tarefas criativas ou analíticas

<Tip>
** Dica Pro:** Comece com todas as Tools desabilitadas para novas ações e, em seguida, adicione apenas o que você precisa. Isso garante um ótimo desempenho e evita o uso inesperado da Tool.
</Tip>
