## Objetivo

Configure padrões Alter, aparência, ditado, permissões e perfil.

## Passos

1. Abra o Alter e pressione o `Cmd+,`.
2. Configurar **Padrões** para o comportamento de acesso rápido.
3. Configurar **Aparência** para visibilidade e tema de entalhe.
4. Configure ** Dictation** para processador, linguagem e substituições.
5. Configurar **Permissões** e conceder acesso macOS necessário.
6. Configure **Perfil** para papel, projetos e estilo de comunicação.
