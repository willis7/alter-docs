Use este guia para adicionar o Alter como provider customizado no TypingMind.

## Abrir configuracao de modelo customizado

No TypingMind, va para:

`Settings` -> `Models` -> `+ Add Custom Model`

## 1) Configuracao basica

Preencha os campos assim:

- `Name`: `Alter`
- `API Type`: `OpenAI Chat Completions API`
- `Endpoint URL`: `https://alterhq.com/api/v1/chat/completions`
- `Model ID`: escolha um dos IDs suportados abaixo
- `Context Length`: valor padrao ou qualquer valor que voce preferir

![Configuracao basica do TypingMind](/images/integrations/typingmind-basic-config.png)

## 2) Autenticacao

Na secao Authentication:

- `Authentication Type`: `No authentication`

Nota: vamos enviar a API key manualmente via Custom Headers no proximo passo.

## 3) Configuracao avancada (obrigatoria)

Expanda `Advanced` no final e adicione um par de Custom Header:

- `Key` (esquerda): `Authorization`
- `Value` (direita): `Bearer <YOUR_API_KEY>`

Substitua `<YOUR_API_KEY>` pela sua API key real do Alter.
Garanta que exista um espaco entre `Bearer` e a key.

![Header de autorizacao no TypingMind](/images/integrations/typingmind-auth-header.png)
