## Checklist de integracao

1. Defina a base URL como `https://alterhq.com/api` (ou `/v1` se necessario)
2. Defina o header de autenticacao como `Authorization: Bearer YOUR_API_KEY`
3. Use model IDs no formato `Provider#Model-name`
4. Execute uma chamada de listagem de modelos antes da primeira requisicao
