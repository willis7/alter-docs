### Como utilizar Local Tools

<Steps>
  <Step title="Abrir o Tools Manager">
Clique nos três pontos em Alter → **Tools Manager**
  </Step>
  <Step title="Habilitar Tools">
Na seção **Local Tools**, procure e habilite Tools (Calendar, Mapas, etc.)
  </Step>
  <Step title="Usar linguagem natural">
Pergunte ao Alter para executar ações:
    - Manda uma mensagem ao John a dizer que estou atrasado.
    - "Adicione uma reunião ao meu calendário para amanhã às 14h"
    - "Mostra-me os meus e-mails não lidos"
  </Step>
</Steps>

<Note>
Activa ou desactiva todas as Tools com o interruptor global no Tools Manager (Configurações > Tools)
</Note>

## Integração x-callback-url

Alter suporta o [x-callback-url](protocolo https://x-callback-url.com/) para comunicação interaplicativo.

### O que é x-callback-url?

Um padrão que permite que os aplicativos acionem ações uns nos outros usando URLs especiais. Passe dados, execute ações e retorne sem problemas.

### Ativando Alter de Outros Apps

<Steps>
  <Step title="Obter o URL">
Em Editor de Ação, selecione uma ação → "URL Callback" seção
  </Step>
  <Step title="Copiar esquema">
Copiar o URL do `alter://` para essa ação
  </Step>
  <Step title="Usar em qualquer lugar">
Colar em Atalhos, Notas, navegador, etc.
  </Step>
</Steps>

** Exemplo:**
```
alter://action/ask-web?input=What+is+Alter+MacOS
```

### Ativando outros aplicativos do Alter

Criar links clicáveis usando os esquemas de URL do aplicativo alvo:

Aplicativo , URL de exemplo , Ação
---------------------------
Ursinho, `bear://x-callback-url/create?title=Demo`, Criar nota
* Coisas * `things://add?title=Task` * Adicionar tarefa *
□ Rascunhos > Criar rascunho
