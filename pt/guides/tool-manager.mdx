## Objetivo

Conecte e gerencie integrações com segurança com etapas claras de solução de problemas.

## Passos

1. Abra as configurações do Alter e vá para integrações ou gerenciador de Tools.
2. Conecte uma integração de cada vez e teste imediatamente.
3. Confirme permissões necessárias e escopos de conta.
4. Crie um prompt de teste simples para verificar o comportamento esperado.
5. Se ocorrerem falhas, reveja a autenticação do provedor e tente novamente.

## Gerenciando muitos Tools

Habilitar muitas Tools aumenta o uso da janela de contexto e impacta o desempenho. Para a maioria dos fluxos de trabalho, use **Flow** - O orquestrador de Tools do Alter que seleciona automaticamente a Tool certa para cada solicitação.

<Card title="Usar Flow para Orquestração Tool" icon="wand-magic-sparkles" href="/guides/use-flow-orchestration">
Flow reduz o uso do contexto agindo como uma única Tool que encaminha solicitações para a integração adequada automaticamente.
</Card>

Em vez de carregar todas as Tools em cada pedido, Flow:
- Analisa sua solicitação e seleciona a Tool certa
- Lida com orquestração multi-passos através de Tools
- Reduz o uso da janela de contexto e melhora o desempenho

Ativar o Flow no Gestor Tool em ** Local Tools > Alter**.

## Solução de problemas

- Re-autenticar conexão de provedor.
- Confirme permissões locais/rede.
- Verificar os limites e quotas específicos da integração.
