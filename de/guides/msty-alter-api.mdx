Nutzen Sie diese Anleitung, um Msty.ai mit Alter API Router zu verbinden.

## Provider-Konfiguration öffnen

Öffnen Sie in Msty.ai den Provider/Modell-Konfigurationsbereich, in dem Sie festlegen:

- API-Endpoint (Base URL)
- API-Key
- Modellauswahl

## 1) API-Key

Fügen Sie Ihren Alter Key in das API-Key-Feld ein:

`sk-...`

Erzeugen Sie einen Key in Alter über `Settings` -> `Router` -> `Alter API Keys`.

## 2) Inference Endpoint

Setzen Sie den Endpoint auf:

```text
https://alterhq.com/api/v1
```

## 3) Modelle laden und auswählen

Klicken Sie in Msty.ai auf **Fetch models**, nachdem Endpoint + API-Key gesetzt sind.

Msty.ai lädt die verfügbaren Alter-Modelle, damit Sie eins aus der Liste auswählen können.
Sie müssen keine Model IDs manuell eingeben.

![Msty.ai Endpoint- und API-Key-Setup](/images/integrations/msty-endpoint-api-key.jpeg)

## Verbindung prüfen

1. Einstellungen in Msty.ai speichern
2. Einen einfachen Prompt senden
3. Wenn es fehlschlägt: Endpoint und API-Key prüfen und erneut auf **Fetch models** klicken
