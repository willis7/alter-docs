Verwenden Sie diesen Leitfaden, um Alter mit anderen Apps unter Verwendung von URL-Schemata zu integrieren.

## Videoanleitung

- https://www.youtube.com/watch?v=X-XJiveQ-Wc

## Einführung

Alter unterstützt x-callback-url, sodass Apps Aktionen in einander auslösen können.
Sie können:

- Alter-Aktionen von externen Apps auslösen
- Aktionen externer Apps von Alter auslösen
- Eingabedaten in URLs übergeben
- Zur ursprünglichen App zurückkehren (abhängig von der Unterstützung der App)

## Funktionsweise

1. Triggern Sie Alter-Aktionen von einer anderen App über `alter://...` URLs.
2. Triggern Sie Aktionen externer Apps von Alter aus unter Verwendung des URL-Schemas dieser App.

## Alter-Aktionen Von Externen Apps Auslösen

1. Öffnen Sie den Aktionseditor.
2. Wählen Sie eine Aktion aus.
3. Kopieren Sie im Abschnitt **URL Callback** die URL aus dem Feld `Allgemein`.
4. Fügen Sie die URL in eine andere App (Browser, Notizen, Shortcuts usw.) ein.

![Alter Callback URLs](/images/integrations/alter-callback-urls-focus.webp)

## Aktionen Externer Apps Von Alter Auslösen

1. Erstellen Sie eine klickbare URL in Alter unter Verwendung des Ziel-App-Schemas.
2. Formatieren Sie die Parameter genau wie von dieser App erforderlich.
3. Klicken Sie, um die Aktion auszuführen und das Verhalten zu überprüfen.

## Beispiele

### Alter-Callback-Beispiele

- Suchanfrage:
  - `alter://action/ask-web?input=What+is+Alter+MacOS`
  - <a href="alter://action/ask-web?input=What+is+Alter+MacOS">In Alter öffnen</a>

- Business-Stratege ohne Eingabe:
  - `alter://action/business-strategist-gpt`
  - <a href="alter://action/business-strategist-gpt">In Alter öffnen</a>

- Business-Stratege mit Eingabe:
  - `alter://action/business-strategist-gpt?input=Explain+Red+Ocean+Strategy`
  - <a href="alter://action/business-strategist-gpt?input=Explain+Red+Ocean+Strategy">In Alter öffnen</a>

### Cross-App-Beispiel

- Bear-Notizenerstellung:
  - `bear://x-callback-url/create?title=Demo%20Note`

## Testen Und Fehlersuche

- Überprüfen Sie das URL-Schema und die Parameter.
- URL-kodieren Sie Sonderzeichen.
- Bestätigen Sie, dass die Ziel-App die Aktion unterstützt.
- Fügen Sie die URL der Callback-Dokumentation der App in Alter ein, um die Syntax/Regeln festzulegen.
