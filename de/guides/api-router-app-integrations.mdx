## Integrations-Checkliste

1. Base URL auf `https://alterhq.com/api` setzen (oder `/v1`, falls erforderlich)
2. Auth-Header auf `Authorization: Bearer YOUR_API_KEY` setzen
3. Model-IDs im Format `Provider#Model-name` verwenden
4. Vor der ersten Anfrage einen Modell-List-Call ausführen
