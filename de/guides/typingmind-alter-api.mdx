Nutzen Sie diese Anleitung, um Alter in TypingMind als benutzerdefinierten Provider hinzuzufügen.

## Benutzerdefiniertes Modell-Setup öffnen

In TypingMind gehen Sie zu:

`Settings` -> `Models` -> `+ Add Custom Model`

## 1) Grundkonfiguration

Setzen Sie die Felder wie folgt:

- `Name`: `Alter`
- `API Type`: `OpenAI Chat Completions API`
- `Endpoint URL`: `https://alterhq.com/api/v1/chat/completions`
- `Model ID`: wählen Sie eine der unterstützten IDs unten
- `Context Length`: Standardwert oder ein beliebiger Wert

![TypingMind Grundkonfiguration](/images/integrations/typingmind-basic-config.png)

## 2) Authentifizierung

Im Bereich Authentication:

- `Authentication Type`: `No authentication`

Hinweis: Den API-Key geben wir im nächsten Schritt manuell über Custom Headers mit.

## 3) Erweiterte Konfiguration (erforderlich)

`Advanced` unten aufklappen und ein Custom Header-Paar hinzufügen:

- `Key` (links): `Authorization`
- `Value` (rechts): `Bearer <YOUR_API_KEY>`

Ersetzen Sie `<YOUR_API_KEY>` durch Ihren echten Alter API-Key.
Achten Sie darauf, dass zwischen `Bearer` und dem Key genau ein Leerzeichen steht.

![TypingMind Authorization Header](/images/integrations/typingmind-auth-header.png)
