---

Verwenden Sie Arbeitsbereiche, wenn einmaliges Löschen von Dateien nicht ausreicht und Sie dauerhaften Kontext benötigen.

## Warum Arbeitsbereiche verwenden?

<CardGroup cols={3}>
  <Card title="Faster responses" icon="zap">
    Halten Sie den relevanten Kontext indiziert und bereit.
  </Card>
  <Card title="Lower cost" icon="credit-card">
    Senden Sie jedes Mal fokussierten Kontext anstelle großer Roheingaben.
  </Card>
  <Card title="Large folders" icon="folder-open">
    Arbeiten Sie mit größeren Codebasen und Dokumentensätzen.
  </Card>
</CardGroup>

## Erstellen Sie Ihren ersten Arbeitsbereich

<Steps>
  <Step title="Open The Hub">
    Drücken Sie `Shift + Cmd + H`.
  </Step>
  <Step title="Go to Workspaces">
    Öffnen Sie den Abschnitt „Arbeitsbereiche“ in der Seitenleiste.
  </Step>
  <Step title="Create workspace">
    Klicken Sie in der Arbeitsbereichsauswahl auf `+`.
  </Step>
  <Step title="Add files and folders">
    Fügen Sie nur die Ressourcen hinzu, die für einen häufig ausgeführten Workflow erforderlich sind.
  </Step>
</Steps>

## Wie Dateien vorbereitet werden

| Dateityp | Konvertierung |
|---|---|
| PDFs | Markdown mit Seitenmarkierungen |
| DOCX | Abschlag |
| XLSX | Markdown oder CSV |
| Bilder | OCR-Textextraktion |
| Verzeichnisse | Rekursive Verarbeitung |

<Note>
  Audio- und Videodateien werden nicht direkt in Arbeitsbereichen unterstützt. Transkribieren Sie sie zunächst, indem Sie sie in Alter ablegen.
</Note>

## Lokaler Speicher

Arbeitsbereichsdaten werden lokal gespeichert in:

```text
~/Library/Application Support/Alter/Workspaces/
```
