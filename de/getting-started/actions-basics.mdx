Verwenden Sie Aktionen, wenn Sie dasselbe Aufgabenmuster wiederholen und eine konsistente Ausgabequalität wünschen.

## Was eine Aktion dir gibt

<CardGroup cols={3}>
  <Card title="Reusable prompts" icon="repeat">
    Führen Sie denselben Workflow aus, ohne die Anweisungen jedes Mal neu schreiben zu müssen.
  </Card>
  <Card title="Model choice" icon="brain">
    Weisen Sie jeder Aufgabe das beste Modell zu.
  </Card>
  <Card title="Tool control" icon="sliders-horizontal">
    Aktivieren Sie nur die Tools, die für sicherere und schnellere Läufe erforderlich sind.
  </Card>
</CardGroup>

## Erstellen Sie Ihre erste Aktion

<Steps>
  <Step title="Open Action Editor">
    Öffnen Sie das Fenstermenü „Ändern“ und wählen Sie **Aktionseditor**.
  </Step>
  <Step title="Create or duplicate">
    Erstellen Sie eine neue Aktion oder duplizieren Sie eine integrierte Aktion, bevor Sie sie anpassen.
  </Step>
  <Step title="Set clear instructions">
    Definieren Sie ein konkretes Ergebnis, z. B. eine wöchentliche Zusammenfassung, Versionshinweise oder eine Sortierung im Posteingang.
  </Step>
  <Step title="Save and run">
    Testen Sie mit einer echten Eingabe und verfeinern Sie den Wortlaut bei Bedarf.
  </Step>
</Steps>

<Note>
  Integrierte Aktionen sind schreibgeschützt. Duplizieren Sie sie, um das Verhalten anzupassen.
</Note>

## Wählen Sie pro Aktion ein Modell und Tools aus

Verwenden Sie **Erweiterte** Einstellungen in der Aktion, um einen Modell- und Werkzeugzugriff zuzuweisen.

- Wählen Sie lokale Modelle für sensible Aufgaben.
- Wählen Sie schnellere Modelle für automatisierungsintensive Aufgaben.
- Beginnen Sie mit minimalen Tools und fügen Sie dann nur das hinzu, was Ihr Workflow benötigt.
