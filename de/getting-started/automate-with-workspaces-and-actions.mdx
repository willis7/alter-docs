Nach den ersten erfolgreichen Aufgaben ist dies der schnellste Weg zum Levelaufstieg.

## Was Ihnen das hilft

Verwenden Sie Arbeitsbereiche, um den Kontext großer Projekte bereitzuhalten, und verwenden Sie dann Aktionen, um hochwertige Aufgaben mit konsistenten Eingabeaufforderungen zu wiederholen.

<CardGroup cols={3}>
  <Card title="Workspaces" icon="folder-tree">
    Halten Sie den Projektkontext bereit, damit Alter schnell mit großen Dateisätzen arbeiten kann.
  </Card>
  <Card title="Actions" icon="zap">
    Speichern Sie wiederverwendbare Eingabeaufforderungen für wiederkehrende Aufgaben und sorgen Sie für eine gleichbleibende Ausgabequalität.
  </Card>
  <Card title="Tool scope" icon="sliders-horizontal">
    Aktivieren Sie für eine höhere Zuverlässigkeit nur die benötigten Tools pro Workflow.
  </Card>
</CardGroup>

## Erstellen Sie Ihren ersten wiederholbaren Workflow

<Steps>
  <Step title="Create one workspace">
    Fügen Sie die Ordner und Dateien für ein einzelnes Projekt hinzu, das Sie jede Woche berühren.
  </Step>
  <Step title="Create one action">
    Verwandeln Sie eine wiederkehrende Aufgabe in eine wiederverwendbare Aktion (z. B. wöchentliche Zusammenfassung oder Versionshinweise).
  </Step>
  <Step title="Constrain tools">
    Aktivieren Sie nur die Tools, die für diese Aktion erforderlich sind, und führen Sie dann den Flow Ende für Ende aus.
  </Step>
</Steps>

<Tip>
  Beginnen Sie mit einem Arbeitsbereich und einer Aktion. Kleine, zuverlässige Automatisierungen lassen sich besser skalieren als große, generische Setups.
</Tip>
