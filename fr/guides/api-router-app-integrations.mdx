## Checklist d'intégration

1. Définissez l'URL de base sur `https://alterhq.com/api` (ou `/v1` si nécessaire)
2. Définissez le header d'authentification sur `Authorization: Bearer YOUR_API_KEY`
3. Utilisez les IDs de modèle au format `Provider#Model-name`
4. Lancez un appel de liste des modèles avant la première requête
