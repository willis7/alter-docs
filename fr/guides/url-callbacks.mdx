Utilisez ce guide pour intégrer Alter à d'autres applications utilisant des schémas d'URL.

## Tutoriel vidéo

- https://www.youtube.com/watch?v=X-XJiveQ-Wc

## Introduction

Alter prend en charge x-callback-url afin que les applications puissent déclencher des actions les unes dans les autres.
Tu peux:

- déclencher des actions Alter à partir d'applications externes
- déclencher des actions d'application externes à partir d'Alter
- transmettre les données d'entrée dans les URL
- revenir à l'application d'origine (en fonction de la prise en charge de l'application)

## Comment ça marche

1. Déclenchez des actions Alter à partir d'une autre application via les URL `alter://...`.
2. Déclenchez des actions d'application externe à partir d'Alter à l'aide du schéma d'URL de cette application.

## Déclencher des actions de modification à partir d'applications externes

1. Ouvrez l'éditeur d'actions.
2. Sélectionnez une action.
3. Dans `General`, copiez l'URL de la section **URL Callback**.
4. Collez l'URL dans une autre application (navigateur, notes, raccourcis, etc.).

![URL Callbacks Alter](/images/integrations/alter-callback-urls-focus.webp)

## Déclencher des actions d'application externes à partir d'Alter

1. Créez une URL cliquable dans Alter à l'aide du schéma d'application cible.
2. Formatez les paramètres exactement comme requis par cette application.
3. Cliquez pour exécuter et vérifier le comportement de l'action.

## Exemples

### Exemples URL Callbacks Alter

- Requête de recherche :
  - `alter://action/ask-web?input=What+is+Alter+MacOS`
  - [alter://action/ask-web?input=What+is+Alter+MacOS](alter://action/ask-web?input=What+is+Alter+MacOS)

- Stratège d'entreprise sans contribution :
  - `alter://action/business-strategist-gpt`
  - [alter://action/business-strategist-gpt](alter://action/business-strategist-gpt)

- Stratège d'affaires avec contribution :
  - `alter://action/business-strategist-gpt?input=Explain+Red+Ocean+Strategy`
  - [alter://action/business-strategist-gpt?input=Explain+Red+Ocean+Strategy](alter://action/business-strategist-gpt?input=Explain+Red+Ocean+Strategy)

### Exemple multi-applications

- Création de notes d'ours :
  - `bear://x-callback-url/create?title=Demo%20Note`

## Tests et dépannage

- Vérifiez le schéma et les paramètres d'URL.
- Caractères spéciaux d’encodage d’URL.
- Confirmez que l'application cible prend en charge l'action.
- Collez l'URL de documentation de l'application dans Alter pour respecter la syntaxe et les règles de base.
