## But

Découvrez comment personnaliser Alter en fonction de vos préférences, gérer votre compte, configurer des modèles d'IA et configurer des autorisations pour une productivité optimale.

## Mesures

### 1. Accéder aux paramètres

Il existe trois façons d'ouvrir les paramètres :

**Option 1 : barre de menus**
Cliquez sur l'icône Alter dans la barre de menu et sélectionnez **Paramètres...** dans le menu déroulant.

**Option 2 : Raccourci clavier**
Appuyez sur **⌘ , (Commande + Virgule)** pour ouvrir directement les paramètres.

**Option 3 : Menu à encoches**
Cliquez sur l'icône de menu à trois points (⋮) dans l'encoche Alter et sélectionnez **Paramètres**.

![Accès à la barre de menus](/images/settings/Alter-Menu_Bar-Menu.webp)

### Paramètres généraux

Dans l'onglet **Général**, vous pouvez configurer les fonctionnalités principales et les options d'accès rapide :

**Accès rapide**
- **Ouvrir l'encoche** : activez le raccourci clavier pour ouvrir l'encoche depuis n'importe où sur votre écran.
- **Ouvrez ou concentrez-vous sur le Hub** : basculez ou lancez rapidement la fenêtre Alter Hub (⌘ ⌥ H)
- **Nouveau chat Hub** : démarrez une nouvelle session de conversation dans le Hub (⌘ Space)
- **Bouton de sélection** : affichez un bouton d'action rapide accessible de n'importe où

**Paramètres LLM**
- **Max Tokens** : définissez la longueur maximale des réponses de l'IA 
- **Température** : contrôlez la créativité et la variation de la réponse (0,0 = déterministe, 1,0 = créatif ; par défaut : 0,70)

**Notifications**
- **Résumer la vidéo YouTube** : créez automatiquement des résumés lorsque vous visitez des vidéos YouTube
- **Enregistrer dans la transcription** : transcrivez automatiquement les enregistrements dans vos notes (accédez aux paramètres Notes pour plus d'options)

![Paramètres généraux](/images/settings/Alter-Settings_01_General-Beta71.webp)

### 2. Paramètres vocaux

L'onglet **Voix** contrôle la manière dont Alter traite et répond à l'audio :

**Configuration générale**

- **Comportement audio pendant la dictée** : contrôlez la manière dont l'audio du système est géré pendant la dictée. Les options incluent l'atténuation (réduire le volume) ou la désactivation complète du son pour minimiser les interruptions
- ** Restez éveillé pendant la réunion ** : empêchez votre Mac de se mettre en veille ou de passer en mode d'économie d'énergie pendant les enregistrements de réunion, garantissant ainsi une transcription ininterrompue
- **Lire un carillon au démarrage de l'enregistrement** : recevez une alerte audio lorsque vous commencez une dictée ou l'enregistrement d'une réunion 

**Langue et vocabulaire**
- **Langue de dictée** : sélectionnez votre langue préférée pour la saisie vocale
- **Dictionnaire personnel** : ajoutez des termes personnalisés et des remplacements de transcription
- **Gérer les remplacements** : modifiez les mots fréquemment mal reconnus

** Processeur vocal **
Choisissez votre modèle de traitement vocal :
- **Apple** : intégré, optimisé pour la confidentialité
- **Local** : exécutez sur votre ordinateur avec les modèles téléchargés
- **Options cloud** : Anthropic et autres fournisseurs cloud avec diverses fonctionnalités

![Paramètres vocaux](/images/settings/Alter-Settings_02_Voice-Beta71.webp)

#### Paramètres vocaux - Raccourcis clavier

Configurez les raccourcis clavier pour un accès vocal rapide depuis n'importe où :

- **Raccourci clavier** : Démarrer/arrêter la saisie vocale et les commandes vocales. Personnalisez la clé d'activation pour un fonctionnement mains libres. Maintenez pour parler à l'invite, appuyez sur pour la dictée.
- **Raccourci de réunion** : accès rapide aux commandes vocales spécifiques à la réunion
- **Live Captions** : Activez la superposition de transcription en direct pendant que vous parlez
- **Live Notepad** : ouvrez le panneau du bloc-notes en direct pour capturer la saisie vocale directement dans les notes.

Ces raccourcis vous permettent d'activer les fonctionnalités vocales d'Alter sans changer de fenêtre ni lâcher le clavier.

![Raccourcis vocaux](/images/settings/Alter-Settings_02a_Voice-Shortcuts-Beta71.webp)

#### Paramètres vocaux - Automatisation

Configurez l'automatisation des réunions et des enregistrements :
- **Applications de réunion** : sélectionnez les applications à surveiller pour les réunions
- **Enregistrement automatique des réunions** : démarrez automatiquement l'enregistrement lorsqu'une réunion est détectée
- **Rechercher les noms des intervenants** : identifiez le nom de chaque intervenant en fonction du service de réunion que vous utilisez
- **Conserver la dictée dans le presse-papier** : conserver la saisie vocale dans le presse-papiers

![Automatisation vocale](/images/settings/Alter-Settings_02b_Voice-Automation-Beta71.webp)

#### Paramètres vocaux - Dictionnaire

Créez un dictionnaire personnel pour une transcription précise :
- Ajouter des mots ou des phrases Alter reconnaît fréquemment des erreurs
- Créez une terminologie personnalisée pour votre domaine ou votre flux de travail

![Dictionnaire vocal](/images/settings/Alter-Settings_02c_Voice-Dictionary-Beta71.webp)

#### Paramètres vocaux - Remplacements

Configurez des remplacements de mots pour les erreurs de transcription courantes :
- Cartographiez les mots fréquemment mal entendus pour les corriger
- Exemple : « Alter » → « Alter » (si vous avez mal entendu « Autel »)

![Remplacements de voix](/images/settings/Alter-Settings_02d_Voice-Replacements-Beta71.webp)

#### Paramètres vocaux - Modèles locaux

Choisissez parmi les modèles de traitement vocal local lorsque vous utilisez le processeur local :

**Pourquoi un traitement local ?**
- **Confidentialité totale** : tout le traitement audio s'effectue sur votre Mac : aucune donnée n'est envoyée aux services cloud
- **Zéro partage de données** : les enregistrements, les transcriptions et l'identification du locuteur restent entièrement sur votre appareil
- **Conforme au RGPD** : conçu selon les principes de protection des données dès la conception
- **Toujours disponible** : fonctionne hors ligne sans dépendance à Internet

**Local** – Options de modèle local de base
- Modèles chuchotés pour la synthèse vocale
- Plusieurs tailles de modèles (Tiny, Base, Small, Medium, Large, etc.)
- Choisissez en fonction du compromis entre précision et performances

**Modèles chuchotés**

OpenAI Whisper prend en charge plus de 50 langues avec une couverture mondiale complète :
- **Langues principales** : anglais, espagnol, français, allemand, portugais, russe, italien, japonais, coréen, chinois
- **Langues européennes** : néerlandais, polonais, roumain, suédois, ukrainien, grec, tchèque, hongrois, slovaque, turc, danois, norvégien, finnois
- **Autres langues** : arabe, hébreu, hindi, vietnamien, thaï, indonésien, tagalog, bengali, tamoul, marathi, kannada et bien d'autres.

**Caractéristiques:**
- Horodatages au niveau des mots générés automatiquement
- Identification du locuteur (qui a dit quoi)
- Optimisé à 626 Mo pour les performances Mac
- Idéal pour la couverture linguistique et la précision mondiales

Whisper excelle dans les langues largement parlées, mais la précision peut varier pour les langues moins courantes avec des données de formation limitées.

![Voix - Modèles locaux](/images/settings/Alter-Settings_02e_Voice-Local-Beta71.webp)

**Local Plus** - Fonctionnalité payante

Comprend les modèles Whisper et Parakeet avec détection automatique de la langue :

**Modèles Whisper** (99 langues)
- Même prise en charge linguistique étendue que l'option locale
- Optimisé pour la précision dans divers contextes linguistiques

**Modèles de perruches** (25 langues européennes)
- Moteur de synthèse vocale rapide et léger de NVIDIA
- **10 x plus rapide que Whisper** pour la transcription des réunions sur Mac
- Détection automatique de la langue à partir de l'audio
- Prend en charge le bulgare, le croate, le tchèque, le danois, le néerlandais, l'anglais, l'estonien, le finnois, le français, l'allemand, le grec, le hongrois, l'italien, le letton, le lituanien, le maltais et plus de 9 autres langues européennes

**Caractéristiques:**
- Diarisation des locuteurs (identifier les différents locuteurs)
- Optimisé à 494 Mo pour les performances Mac
- Idéal pour l'enregistrement de réunions en temps réel
- Idéal pour les charges de travail linguistiques européennes

**Choisissez votre modèle :**
- **Perruche** : traitement plus rapide, utilisation réduite des ressources, idéal pour les réunions (25 langues européennes)
- **Whisper** : couverture linguistique plus large, précision maximale (plus de 50 langues mondiales)

Les deux modèles fournissent un horodatage au niveau des mots, une identification du locuteur et une détection automatique de la langue, le tout traité localement sur votre Mac.

![Voix - Modèles locaux Plus](/images/settings/Alter-Settings_02f_Voice-Local_Plus-Beta71.webp)

### 3. Paramètres du profil

L'onglet **Profil** gère votre identité et vos préférences :

**E-mail du compte**
- Afficher l'e-mail de votre compte actuel
- Utilisé pour la facturation et la récupération de compte

**Intégration Discord**
- Connectez ou déconnectez votre compte Discord
- Permet des flux de travail spécifiques à Discord

**Mon profil**
- Ajouter une description de profil (aide Alter à fournir des réponses mieux adaptées)
- Informations personnalisées Alter utilise dans les conversations

**Langage de génération**
- Définissez votre langue préférée pour les réponses d'Alter (par défaut : anglais)
- Remarque : Cela peut varier en fonction du modèle que vous utilisez

![Paramètres du profil](/images/settings/Alter-Settings_03_Profile-Beta71.webp)

### 4. Paramètres d'apparence

L'onglet **Apparence** contrôle l'apparence visuelle :

**Options de thème**
- **Système** : faites correspondre l'apparence du système de votre Mac
- **Lumière** : thème Forcer la lumière
- **Sombre** : Thème sombre forcé

**Personnalisation**
- **Position du panneau par défaut** : là où les panneaux apparaissent (en haut à droite par défaut)
- **Taille de police du menu** : Ajustez la taille du texte pour plus de lisibilité (par défaut : 16)
- **Taille de police du message** : définir la taille du texte du message de discussion (par défaut : 15)
- **Encoche d'affichage** : Afficher l'indicateur d'état
- **Afficher le bloc de contrôle dans la barre de menu** : Afficher l'indicateur dans la barre de menu Alter
- **Masquer dans les captures d'écran** : exclure Alter des captures d'écran et du partage d'écran 

![Paramètres d'apparence - Thème sombre](/images/settings/Alter-Settings_04a_Appearance-Dark-Beta71.webp)

![Paramètres d'apparence - Thème clair](/images/settings/Alter-Settings_04b_Appearance-Light-Beta71.webp)

![Paramètres d'apparence - Vue complète](/images/settings/Alter-Settings_04c_Appearance-Full-Beta71.webp)

### 5. Paramètres de facturation

L'onglet **Facturation** affiche l'état de votre abonnement :

**Informations sur la licence**
- Votre forfait actuel (par exemple, Pro Plan)
- Date d'expiration
- Alter ou mettre fin aux options d'abonnement

**Utilisation équitable**
- Indicateur visuel de votre allocation d'utilisation équitable restante
- Affiche la chronologie de la réinitialisation (par exemple, "Réinitialiser dans 20 jours")

![Paramètres de facturation](/images/settings/Alter-Settings_05_Billing-Beta71.webp)

### 6. Paramètres d'autorisations

L'onglet **Autorisations** gère l'accès au système et les préférences de collecte de données :

**Autorisations système**
Chaque autorisation peut être activée avec des boutons d'accès rapide :
- **Lancer à la connexion** : démarrez Alter automatiquement au démarrage de votre Mac
- **Accessibilité** : requis pour les commandes clavier et le contrôle des fenêtres (vérifier l'accessibilité)
- **Préférences Système** : Autoriser l'accès aux paramètres système (vérifier les préférences système)
- **Enregistrement d'écran** : capturez le contenu de l'écran pour l'analyse et la prise en compte du contexte (vérifier l'enregistrement d'écran)
- **Microphone** : Activer la saisie vocale et la dictée (Vérifier le microphone)

**Analytique**
- **Désactiver Analytics** : contrôlez si Alter collecte des données d'utilisation anonymisées. Lorsqu'il est activé, Alter collecte uniquement les interactions anonymes de l'interface utilisateur et les données d'erreur pour améliorer le produit : aucune donnée personnelle ni aucun contenu n'est suivi.

![Paramètres d'autorisations](/images/settings/Alter-Settings_06_Permissions-Beta71.webp)

### 7. Paramètres des clés API

L'onglet **Clés API** vous permet d'apporter vos propres modèles :

**Partenaires**
- Intégrez des services partenaires qui fournissent des points de terminaison compatibles OpenAI

**Fournisseurs personnalisés**
Ajoutez et gérez des fournisseurs LLM personnalisés :
- **Ajouter un fournisseur** : créer un nouveau point de terminaison de modèle personnalisé
- **URL du fournisseur** : URL du point de terminaison (par exemple, `http://localhost:1234/v1`)
- **Gérer les modèles** : affichez et testez les modèles disponibles auprès de chaque fournisseur

**Types de fournisseurs pris en charge**
Lors de l'ajout d'un fournisseur personnalisé, vous pouvez choisir parmi :
- **OpenAI** : API d'OpenAI (modèles GPT)
- **Personnalisé** : points de terminaison génériques compatibles OpenAI
- **Azure** : service Microsoft Azure OpenAI
- **LM Studio** : modèles locaux via LM Studio
- **PrivateMode** : moteurs d'inférence locaux
- **Bifrost** : fournisseur d'API Bifrost
- **Mistral** : modèles Mistral IA
- **Gemini** : API Google Gemini
- **OpenRouter** : service de routeur multimodèle
- **Ollama** : cadre LLM local
- **Osaurus** : fournisseur d'Osaurus

**Ajout d'un fournisseur**
1. Cliquez sur le bouton « Ajouter un fournisseur »
2. Sélectionnez le type de fournisseur dans la liste déroulante
3. Saisissez l'URL du point de terminaison de l'API
4. Fournissez votre clé API
5. Cliquez sur "Enregistrer" pour configurer

### 8. Paramètres du routeur

L'onglet **Routeur** configure le point de terminaison AI par défaut :

**Point de terminaison compatible OpenAI**
- Point de terminaison par défaut pour alimenter les modèles Alter
- Lien : https://alterhq.com/api

**clés API Alter**
- Générez et gérez vos clés API personnelles
- Utilisation pour les intégrations et les Tools tiers
