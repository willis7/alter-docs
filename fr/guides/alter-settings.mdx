## Aperçu


La fenêtre les paramètres d'Alter offre un contrôle complet sur les fonctionnalités, l'apparence et les capacités d'intégration de l'application. 

### Accéder aux modifications des paramètres

Lorsque la fenêtre Alter est visible


1. Utilisez le raccourci clavier lorsque la fenêtre Alter est ouverte : `cmd+,`

ou 

2. cliquez sur les 3 points de la fenêtre principale /

### Valeurs par défaut

![les paramètres d'Alter par défaut](https://beehiiv-images-production.s3.amazonaws.com/uploads/asset/file/12283a80-42d2-4b7e-a3f6-bb98c09af825/Alter_Settings_Default.png?t=1740500278)

Paramètres d'accès rapide :

- Combinaison de touches à encoche ouverte (par défaut : `Alt+Space`)
- Bouton de sélection pour un accès rapide lors de la sélection de texte

Paramètres LLM :
- Max Tokens : définir la longueur maximale de génération de texte (par défaut : 2 048)
- Température : Ajuster de déterministe à créatif (par défaut : 0,70)

### Dictée 

![Paramètres de dictée](/images/settings/dictation.webp)


#### Raccourci

**Reliure de touches globale**
Définissez un raccourci clavier personnalisé pour déclencher les fonctionnalités vocales d'Alter. Si elle est laissée vide, la touche Fn sera utilisée par défaut.

**Comportement:**
- **Maintenir** - Active le mode parler-à-commander
- **Un clic** - Démarre le mode dictée

** Raccourci d'enregistrement **
Cliquez sur "Enregistrer le raccourci" pour capturer votre combinaison de touches préférée.

#### Gestion des fichiers

**Répertoire des relevés de notes**
Accédez à vos transcriptions et enregistrements enregistrés via le Finder. Cliquez sur "Ouvrir dans le Finder" pour accéder directement à l'endroit où Alter stocke vos données vocales.

#### Langue

**Langage de dictée**
Définissez la langue principale pour la reconnaissance vocale et la transcription. Lorsqu'il est défini sur "Automatique", Alter tentera de détecter la langue parlée.

**Remarque :** Toutes les fonctionnalités vocales utiliseront ce paramètre de langue pour traiter les transcriptions, y compris les réunions.

#### Automation

** Réunions d'enregistrement automatique **
Lorsqu'il est activé, Alter démarre automatiquement l'enregistrement lorsqu'il détecte que vous êtes en réunion.

**Seuil d'atteinte**
Définissez la durée d'un enregistrement (en secondes) avant qu'Alter le considère comme une réunion plutôt que comme une note vocale rapide. La valeur par défaut est 120 secondes (2 minutes).

**Conserver la dictée dans le carton**
Lorsqu'il est activé, le texte dicté est automatiquement copié dans votre presse-papiers/pâte-papier, ce qui facilite son collage dans n'importe quelle application.

#### Dictionnaire

**Reconnaissance de mots personnalisée**
Ajoutez des mots que vous souhaitez qu'Alter reconnaisse mieux lors de la transcription. Ceci est particulièrement utile pour :
- Termes techniques spécifiques à votre secteur d'activité
- Noms de sociétés et noms de produits
- Noms propres qui pourraient être mal compris
- Acronymes et abréviations

**Format :** Saisissez des mots séparés par des virgules dans le champ de texte.

#### Processeur

**Moteur de traitement de la parole**
Choisissez comment Alter traite votre saisie vocale :

1. **Cloud** : utilise le traitement basé sur le cloud pour une précision maximale
2. **Local (Apple)** – utilise la reconnaissance vocale intégrée à l'appareil d'Apple
3. **Local (Whisper)** – utilise le modèle Whisper d'OpenAI localement pour garantir la confidentialité
4. **Whisper Pro** - Traitement local avancé avec transcriptions de 10 secondes pour les réunions

---

*Ces paramètres vous aident à personnaliser les fonctionnalités de reconnaissance vocale et d'automatisation d'Alter en fonction de votre flux de travail et de vos préférences.*

### Profil 

![Alter le profil des paramètres](https://beehiiv-images-production.s3.amazonaws.com/uploads/asset/file/cecd74bb-f332-4c17-a8e1-f71cb35cabff/Alter_Settings_Profile.png?t=1740500288)

Le profil est un moyen simple de dire à Alter qui vous êtes et comment vous souhaitez travailler ensemble. Le contexte est important : lorsque Alter en sait plus sur vous, il peut éviter les futilités et fournir des réponses plus utiles et ciblées.

**Mon profil :**
Utilisez cette section de personnalisation pour :
- **Parlez de vous à Alter** : partagez votre rôle, votre expertise et votre expérience
- **Partagez ce sur quoi vous travaillez** : décrivez vos projets et objectifs actuels  
- **Définissez votre style de communication préféré** : précisez la manière dont vous souhaitez recevoir des informations
- **Ajoutez des explications pour les Tools** : détaillez la manière dont vous souhaitez que certains Tools soient utilisés et fournissez des références techniques si nécessaire (paires de noms et d'identifiants par exemple)

**Pourquoi le profil est important :**
Lorsqu'Alter sait que vous êtes un développeur qui préfère des réponses techniques concises, ou un écrivain qui souhaite un brainstorming créatif, il peut adapter ses réponses en conséquence. Cette personnalisation aide Alter à comprendre votre contexte et à fournir une assistance plus pertinente.

Voici un exemple simple :

```
My name is Samuel ROY, I'm the CEO of Alter, an AI assistant that understands your Mac. It's the first AI assistant living in the macOS notch.  I bring vision and manage product development, I can code and do marketing tasks. I like my AI to output content in a genuine, friendly and direct tone. Humor and Internet culture is appreciated. 

When you use tools, you might need the following informations:
<DISCORD>
Guild ID: XXXXXXX
General channel: general
Private team channel: XXXXX
</DISCORD>

<GITHUB>
Alter app repo: XXXXX
Alter server: XXX
Alter website (alterhq.com): XXXX
</GITHUB>

```


- Langue de génération : sélectionnez la langue préférée pour les sorties
Remarque : La sélection de la langue affecte uniquement la génération, pas l'interface de l'application.

#### Autorisations requises

Alter fonctionnera au mieux lorsque ces autorisations seront activées

- Autorisations d'accessibilité
   - Objectif : utilisé pour obtenir le texte sélectionné dans tout le système et pour coller du texte n'importe où
   - Statut : OK
   - Bouton d'action : "Vérifier l'accessibilité"

- Autorisations d'événements système
   - Objectif : utilisé pour interagir avec d'autres applications
   - Statut : OK
   - Bouton d'action : "Vérifier les événements système"

- Autorisations d'enregistrement d'écran
   - Objectif : utilisé pour capturer le contenu de l'écran
   - Statut : OK
   - Bouton d'action : "Vérifier l'enregistrement d'écran"

- Autorisations du microphone
   - Objectif : utilisé pour enregistrer l'entrée audio
   - Statut : OK
   - Bouton d'action : "Vérifier le microphone"

#### Désactiver les analyses

Nous nous appuyons sur l'analyse pour améliorer Alter. Cependant vous êtes libre de le désactiver si vous le préférez

- **Désactiver l'analyse** : basculez le commutateur pour contrôler la collecte d'analyses.
