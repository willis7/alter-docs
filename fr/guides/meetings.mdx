## Enregistrement automatique des réunions

### Détection intelligente des réunions

Alter détecte automatiquement lorsque vous rejoignez des réunions sur :

<CardGroup cols={4}>
  <Card title="Zoom" icon="video">
    Détection instantanée
  </Card>
  <Card title="Google Rencontre" icon="google">
    Reconnaissance automatique
  </Card>
  <Card title="Équipes Microsoft" icon="users">
    Prise en charge complète
  </Card>
  <Card title="Autres plateformes" icon="more-horizontal">
    La plupart des applications de conférence
  </Card>
</CardGroup>

Lorsqu'une réunion est détectée, Alter affiche une notification subtile vous donnant la possibilité de l'enregistrer ou de l'ignorer.

### Paramètres d'enregistrement automatique

<Steps>
  <Step title="Activer l'enregistrement automatique">
    Paramètres → Dictée → **Enregistrement automatique des réunions**
  </Step>
  <Step title="Définir le seuil">
    Définir la durée minimale (par défaut : 120 secondes)
  </Step>
  <Step title="Configurer les notifications">
    Choisissez de sauter ou d'enregistrer via notification
  </Step>
  <Step title="Activer le maintien éveillé">
    Empêcher Mac de dormir pendant de longues réunions
  </Step>
</Steps>

<Frame>
  ![Notification d'enregistrement automatique](/images/meetings/auto-record.webp)
</Frame>

### Fonctionnalité de maintien de l'éveil

Lorsqu'il est activé dans **Paramètres > Voix**, Alter empêche votre Mac de se mettre en veille pendant les enregistrements :

<CardGroup cols={2}>
  <Card title="Automatique" icon="zap">
    S'engage lorsque l'enregistrement démarre
  </Card>
  <Card title="Écran actif" icon="monitor">
    Assure un enregistrement ininterrompu
  </Card>
  <Card title="Version intelligente" icon="unlock">
    Se libère automatiquement une fois terminé
  </Card>
  <Card title="Basculer le contrôle" icon="toggle-right">
    Activer/désactiver selon les besoins
  </Card>
</CardGroup>

<Info>
  **Parfait pour :** Les longues conférences téléphoniques, les webinaires, les présentations où la veille de l'écran interromprait l'enregistrement
</Info>

### Enregistrement manuel

Vous pouvez également contrôler manuellement les enregistrements :

<Steps>
  <Step title="Commencer l'enregistrement">
    Cliquez sur l'icône du micro sur l'encoche – elle se transforme en bouton d'enregistrement rouge
  </Step>
  <Step title="Moniteur">
    Le micro s'anime pendant la transcription
  </Step>
  <Step title="Arrêt">
    Cliquez sur le bouton rouge pour terminer
  </Step>
</Steps>

## Sous-titres en direct

Live Captions apporte une transcription en temps réel à chaque application de votre Mac, comme les sous-titres YouTube, mais à l'échelle du système.

  <div className="embed-container">

### Comment ça marche

Les sous-titres apparaissent instantanément lors de la lecture audio via n'importe quelle application :

<AccordionGroup>
  <Accordion icon="video" title="Compatibilité universelle">
    Fonctionne avec Zoom, Google Meet, Teams, Safari, podcasts, streaming – n'importe quelle application Mac
  </Accordion>
  
  <Accordion icon="globe" title="Détection automatique de la langue">
    Reconnaît et transcrit automatiquement plusieurs langues
  </Accordion>
  
  <Accordion icon="shield" title="100% local">
    Toutes les transcriptions ont lieu sur votre Mac pour une confidentialité totale
  </Accordion>
</AccordionGroup>

### Utiliser les sous-titres en direct

**Basculer :** Appuyez sur `Cmd + Shift + ↓` pour afficher/masquer les sous-titres.

**Exigences:**
<Steps>
  <Step title="Allez dans Paramètres">
    Accédez à **Paramètres > Voix**.
  </Step>
  <Step title="Sélectionnez le processeur">
    Choisissez **Pro local**
  </Step>
  <Step title="Télécharger le modèle">
    Obtenez **Parakeet V3** (recommandé pour de meilleures performances)
  </Step>
</Steps>

## Stockage de fichiers

### Emplacement

Tous les enregistrements et transcriptions sont stockés localement à :

```
~/Library/Application Support/Alter/Transcripts
```

<Info>
  **100 % privé :** Tout reste sur votre Mac. Pas de stockage cloud, sauf si vous choisissez explicitement le traitement cloud.
</Info>

### Options de gestion

<CardGroup cols={3}>
  <Card title="Accès au Finder" icon="folder">
    Ouvrir le dossier directement depuis les paramètres
  </Card>
  <Card title="Recherche intégrée" icon="search">

  <Card title="Exporter" icon="download">
