## Local Tools

Alter fournit **17+ Local Tools puissants** qui s'intègrent directement à vos applications Mac. Contrairement aux autres assistants IA, Alter contrôle votre Mac via le langage naturel.

### Comment utiliser les Local Tools

<Steps>
  <Step title="Tools Manager ouverts">
    Cliquez sur les trois points dans Alter → **Tools Manager**
  </Step>
  <Step title="Activer les Tools">
    Dans la section **Local Tools**, recherchez et activez les Tools (Calendrier, Cartes, etc.)
  </Step>
  <Step title="Utiliser le langage naturel">
    Demandez à Alter d'effectuer des actions :
    - "Envoyez un message à John pour lui dire que je suis en retard"
    - "Ajouter une réunion à mon agenda pour demain à 14h00"
    - "Montre-moi mes emails non lus"
  </Step>
</Steps>

<Note>
  Activez/désactivez tous les Tools à l'aide du commutateur global dans le Tools Manager (Paramètres > Tools)
</Note>

## Intégration x-callback-url

Alter prend en charge le protocole [x-callback-url](https://x-callback-url.com/) pour la communication inter-applications.

### Qu'est-ce que x-callback-url ?

Une norme qui permet aux applications de déclencher des actions les unes sur les autres à l'aide d'URL spéciales. Transmettez des données, effectuez des actions et revenez en toute transparence.

### Déclenchement de la modification à partir d'autres applications

<Steps>
  <Step title="Obtenez l'URL">
    Dans l'éditeur d'actions, sélectionnez une action → section "URL Callback"
  </Step>
  <Step title="Schéma de copie">
    Copiez l'URL `alter://` de cette action
  </Step>
  <Step title="Utiliser n'importe où">
    Collez dans les raccourcis, les notes, le navigateur, etc.
  </Step>
</Steps>

**Exemple:**
```
alter://action/ask-web?input=What+is+Alter+MacOS
```

### Déclenchement d'autres applications à partir d'Alter

Créez des liens cliquables à l'aide de schémas d'URL d'application cible :

| Application | Exemple d'URL | Actions |
|-----|-------------|--------|
| Ours | `bear://x-callback-url/create?title=Demo` | Créer une note |
| Choses | `things://add?title=Task` | Ajouter une tâche |
| Brouillons | `drafts://x-callback-url/create?text=Hello` | Créer un brouillon |

<Card title="Applications prises en charge" icon="check-circle">
  Bear, Things, Drafts, OmniFocus, 1Writer, Agenda et des centaines d'autres
</Card>
