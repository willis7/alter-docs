---

---

Utilisez des espaces de travail lorsque les suppressions de fichiers ponctuelles ne suffisent pas et que vous avez besoin d’un contexte persistant.

## Pourquoi utiliser les espaces de travail

<CardGroup cols={3}>
  <Card title="Faster responses" icon="zap">
    Gardez le contexte pertinent indexé et prêt.
  </Card>
  <Card title="Lower cost" icon="credit-card">
    Envoyez à chaque fois un contexte ciblé au lieu de grandes entrées brutes.
  </Card>
  <Card title="Large folders" icon="folder-open">
    Travaillez avec des bases de code et des ensembles de documents plus volumineux.
  </Card>
</CardGroup>

## Créez votre premier espace de travail

<Steps>
  <Step title="Open The Hub">
    Appuyez sur `Shift + Cmd + H`.
  </Step>
  <Step title="Go to Workspaces">
    Ouvrez la section Espaces de travail dans la barre latérale.
  </Step>
  <Step title="Create workspace">
    Cliquez sur `+` dans le sélecteur d'espace de travail.
  </Step>
  <Step title="Add files and folders">
    Ajoutez uniquement les ressources nécessaires à un flux de travail que vous exécutez souvent.
  </Step>
</Steps>

## Comment les fichiers sont préparés

| Type de fichier | Transformation |
|---|---|
| PDF | Markdown avec des marqueurs de page |
| DOCX | Démarquage |
| XLSX | Markdown ou CSV |
| Images | Extraction de texte OCR |
| Annuaires | Traitement récursif |

<Note>
  Les fichiers audio et vidéo ne sont pas pris en charge directement dans les espaces de travail. Transcrivez-les d'abord en les déposant dans Alter.
</Note>

## Stockage local

Les données de l'espace de travail sont stockées localement dans :

```text
~/Library/Application Support/Alter/Workspaces/
```
