L'utilisation de l'ordinateur donne à Alter une couche d'exécution d'interface utilisateur pour les flux de travail de bureau. Il peut inspecter les fenêtres, interagir avec les éléments de l'interface utilisateur et rythmer les flux en plusieurs étapes.

## Ensemble d'outils

- [Obtenir le contexte de l'application active](/apps-tools/computer-use-get-active-app-context)
- [Effectuer](/apps-tools/computer-use-perform)
- [Attendez](/apps-tools/computer-use-wait)
- [Défilement](/apps-tools/computer-use-scroll)

## Débit typique

1. Lisez l'état actuel de l'interface utilisateur avec **Get Active App Context**.
2. Déclenchez des actions de l'interface utilisateur avec **Perform**.
3. Utilisez **Scroll** lorsque l'élément cible est hors écran.
4. Ajoutez **Attendez** entre les étapes de chargement ou de transitions d'état.
