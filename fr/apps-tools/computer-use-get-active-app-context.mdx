title: "Obtenir le contexte d'application actif"
description: "Capturez le contenu de la fenêtre active et le texte OCR pour les flux de travail d'utilisation de l'ordinateur"
---

Utilisez cet outil pour lire ce qui est actuellement visible dans une application ou une fenêtre de navigateur avant d'agir.

## Ce que ça fait

- Récupère le contenu de la fenêtre de l'application active par défaut.
- Accepte facultatif `windowID` (`pid/title`) pour cibler une fenêtre spécifique.
- Renvoie le contenu du texte extrait.
- Ajoute du texte OCR à partir de captures d'écran lorsque cela est utile.

## Utilisations courantes

- Lisez l'état de la page ou de l'application avant de cliquer.
- Extrayez le contenu visible pour les résumés.
- Créez un contexte stable avant l'automatisation en plusieurs étapes.

## Saisir

- `windowID` (facultatif) : ciblez une fenêtre spécifique depuis `list_windows`.
