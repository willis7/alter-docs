Cette référence répertorie tous les modèles d'IA disponibles via le routeur Alter API. Utilisez ces noms de modèles lorsque vous effectuez des requêtes API.

## Format de dénomination du modèle

Tous les modèles suivent le format : `Provider#Model-name`

Par exemple : `OpenAI#gpt-5`, `Claude#claude-sonnet-4-6`

## Exemples d'utilisation

### Python
```python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://alterhq.com/api/v1"
)

# Use GPT-5
response = client.chat.completions.create(
    model="OpenAI#gpt-5",
    messages=[{"role": "user", "content": "Hello!"}]
)
```

### boucle
```bash
curl https://alterhq.com/api/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "OpenAI#gpt-5",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```
