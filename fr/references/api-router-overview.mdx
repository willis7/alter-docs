L'Alter API Router est une passerelle API compatible OpenAI qui fournit un accès unifié à plus de 92 modèles d'IA provenant de plus de 10 fournisseurs via un seul point de terminaison.

## Qu'est-ce que le routeur API ?

Le routeur API est un service centralisé qui élimine le besoin de gérer plusieurs clés API et comptes de facturation entre différents fournisseurs d'IA. Au lieu de conserver des comptes séparés avec OpenAI, Anthropic, Google et autres, vous pouvez utiliser Alter comme point d'entrée unique pour tous les accès aux modèles d'IA.

### Avantages clés

- **Clé API unique** pour tous les fournisseurs
- **Facturation centralisée** via votre compte Alter
- **92+ modèles** de plus de 10 fournisseurs
- **Point de terminaison compatible OpenAI** fonctionne avec les Tools et SDK existants
- **Changement de modèle facile** sans changement de code

## Démarrage rapide

### 1. Générez une clé API

1. Ouvrez les **Paramètres** (**⌘ ,**)
2. Accédez à l'onglet **Routeur**
3. Sous "clés API Alter", cliquez sur **Ajouter une nouvelle clé**.
4. Copiez votre clé (commence par `sk-`)

### 2. Obtenez le point de terminaison

```
https://alterhq.com/api
```

Pour la plupart des Tools, utilisez :
```
https://alterhq.com/api/v1
```

### 3. Liste des modèles disponibles

```bash
curl https://alterhq.com/api/models \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Format de dénomination du modèle

Les modèles utilisent le format : `Provider#Model-name`

Exemples :
- `OpenAI#gpt-5` – Dernier GPT-5
- `OpenAI#gpt-5-mini` - GPT-5 léger
- `Claude#claude-sonnet-4-6` - Dernière Claude
- `Gemini#gemini-2.5-pro` - Derniers Gemini

## Exemple d'utilisation

```python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://alterhq.com/api/v1"
)

response = client.chat.completions.create(
    model="OpenAI#gpt-5",
    messages=[
        {"role": "user", "content": "What is machine learning?"}
    ]
)

print(response.choices[0].message.content)
```

## Limites d'utilisation

- **Limite quotidienne** : 200 requêtes par jour dans le cadre d'une utilisation équitable
- **Limitation** : après 200 requêtes, limitation à 1 toutes les 10 minutes.
- **Complément de budget** : disponible pour un accès cohérent au-delà de l'utilisation équitable
