### Navigation globale

Disponible dans toutes les fenêtres (sauf si la fenêtre cible est déjà active).

- **Maj + Cmd + H** : ouvrir Alter Hub
- **Maj + Cmd + E** : Ouvrir l'éditeur d'actions
- **Maj + Cmd + T** : ouvrir le Tools Manager
- **Alt + Espace** (par défaut) : Encoche ouverte – configurable dans **Paramètres > Valeurs par défaut > Liaison de touches encoche ouverte**
- **Raccourci Alter Quick Menu** : ouvrez Alter depuis n'importe où — configurable dans **Paramètres > Général > Accès rapide**
- **Touche de raccourci parole-texte** : démarrer la saisie vocale – configurable dans **Paramètres > Dictée > Raccourci** (par défaut : touche Fn)

### Panneau rapide

Lorsque le panneau rapide est ouvert (depuis l'encoche) :

- **←**, **↑**, **→**, **↓** : Naviguer entre les objets
- **Entrer** : Sélectionnez un objet

### Section Contexte

- **←**, **↑**, **→**, **↓** : Naviguer entre les contextes
- **Barre d'espace** : ajouter/supprimer le(s) contexte(s) sélectionné(s)
- **Cmd + K** : Supprimez tous les contextes de la conversation active
- **Cmd + V** : collez le contenu du presse-papiers pour l'ajouter au contexte

### Boîte d'invite

- **Entrez** : invitez votre modèle actif
- **Cmd + Entrée** : invite à l'aide de la recherche sur le Web (via Gemini Web)
- **Maj + Entrée** : saut de ligne
- **@** : Ajouter des éléments au contexte
- **#** : Afficher l'historique des conversations Alter
- ***** : Afficher l'historique des transcriptions des réunions
- **/** : Listez et choisissez votre modèle
- **?** : Afficher le menu d'aide

### Alter la fenêtre du hub

- **Cmd + F** : recherche ciblée (touches fléchées pour parcourir les résultats)
- **Cmd + B** : basculer la visibilité de la barre latérale
- **Cmd + T** : Nouvel onglet (démarrer une nouvelle discussion)
- **Cmd + J** : Créer un Workspace (nécessite un contexte actif)
- **Cmd + Opt + O** : Basculer vers l'Workspace...
- **Cmd + K** : effacer tous les contextes (nécessite des contextes actifs)
- **Cmd + Supprimer** : Supprimer la conversation (nécessite une conversation active)
- **Cmd + W** : Fermer l'onglet (ferme la fenêtre si c'est le dernier onglet)
- **Cmd + Opt + W** : fermez tous les onglets (nécessite >1 onglet)
- **Cmd + Opt + ←** : onglet précédent (nécessite >1 onglet)
- **Cmd + Opt + →** : onglet suivant (nécessite >1 onglet)
- **Cmd + 1...9** : passer à un index d'onglet spécifique

#### Workspace (nécessite l'Workspace sélectionné)

Aucun raccourci par défaut ; ceux-ci peuvent être définis via les paramètres système macOS :

- Ouvrir les paramètres de l'Workspace
- Renommer l'Workspace
- Définir les autorisations
- Basculer le bac à sable
- Démarrer/Arrêter le conteneur sandbox
- Terminal ouvert dans un bac à sable

### Fenêtre de l'éditeur d'actions

- **Cmd + F** : recherche ciblée
- **Cmd + N** : Nouvelle action
- **Cmd + D** : action en double (nécessite l'action sélectionnée)
- **Cmd + E** : action d'exportation (nécessite l'action sélectionnée)
- **Cmd + O** : action d'importation
- **Cmd + Supprimer** : action de suppression (nécessite l'action sélectionnée)
- **Cmd + W** : Fermer la fenêtre
- **Cmd + [** : section précédente
- **Cmd + ]** : section suivante

### Fenêtre du Tools Manager

- **Cmd + W** : Fermer la fenêtre
- **Cmd + F** : recherche ciblée
- **Lorsque « Compétences » est sélectionné** : Cmd + O (Importer la compétence), Cmd + R (Actualiser la liste)
- **Lorsque « Serveurs MCP distants » est sélectionné** : Cmd + N (Ajouter un serveur), Cmd + R (Actualiser la liste)
- **Lorsque « Actions rapides » est sélectionné** : Cmd + N (Nouvelle action rapide), Supprimer (Supprimer l'action rapide, nécessite une sélection)

### Panneaux de réunions en direct

Disponible lors des réunions uniquement :

- **Cmd + Shift + ↓** : activer/désactiver les sous-titres en direct
- **Maj + Cmd + →** : activer/désactiver le bloc-notes en direct

### les paramètres d'Alter

- **Cmd + ,** : Ouvrir les paramètres (lorsque la fenêtre Alter est visible)
