## Comment ouvrir Alter ?

1. Passez la souris sur l'encoche.
2. Appuyez sur `Alt+Space`.
3. Maintenez `Fn` et parlez.

![Ouvrir Alter](/images/getting-started/open-notch.gif)

Voir [Commencer](/fr/guides/getting-started) pour plus de détails.

## Comment ajouter ma propre clé API ?

1. Ouvrez les paramètres (`Cmd+,`).
2. Accédez aux clés API.
3. Sélectionnez le fournisseur et entrez la clé.

![Clés API personnalisées](/images/models/custom-api-keys.webp)

Voir [Comment : utilisez votre propre clé API](/fr/how-to/use-byok) pour des instructions détaillées.

## Comment ajouter du contexte à une invite ?

- Faites glisser et déposez les fichiers sur l'encoche.
- Cliquez avec le bouton droit sur les fichiers et choisissez Ajouter à modifier.
- Utilisez `@` pour ajouter du contexte à partir de l'invite.

![Glisser et déposer](/images/context/drag-and-drop.gif)

Voir [La fenêtre de modification](/fr/guides/alter-window#the-context-section) pour plus de détails.

## Comment modifier les paramètres rapidement ?

- Ouvrez Alter et appuyez sur `Cmd+,`.
- Ou ouvrez le menu à trois points -> Paramètres.

![Menu Paramètres](/images/settings/settings-menu.png)

Voir [les paramètres d'Alter](/fr/guides/alter-settings) pour plus de détails.

## Comment configurer la dictée ?

Paramètres -> Dictée, puis choisissez le processeur, la langue et les remplacements.

![Dictée](/images/settings/dictation.webp)

Voir [Guide de dictée](/fr/guides/dictation) pour les instructions de configuration complètes.
