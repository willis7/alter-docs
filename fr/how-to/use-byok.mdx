## Mesures

1. Ouvrez les paramètres avec `Cmd+,`.
2. Ouvrez les clés API.
3. Sélectionnez le fournisseur et collez la clé.
4. Tapez `/` dans l'invite pour confirmer l'apparition des modèles personnalisés.
