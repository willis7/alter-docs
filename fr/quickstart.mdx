## Commencez en trois étapes

Commencez à utiliser Alter pour augmenter votre productivité avec l'assistance de l'IA.

### Étape 1 : Télécharger et installer

<AccordionGroup>
  <Accordion icon="download" title="Télécharger Alter">
    Téléchargez Alter depuis notre [site web](https://alterhq.com/beta) ou le Mac App Store.
    
    Configuration système requise :
    - macOS 15.0 ou version ultérieure
    - Apple Silicon ou Intel Mac
  </Accordion>
  <Accordion icon="gear" title="Configuration complète">
    1. Ouvrez le DMG téléchargé et faites glisser Alter vers votre dossier Applications
    2. Lancer Alter depuis les applications
    3. Accordez les autorisations requises lorsque vous y êtes invité (accessibilité, microphone, enregistrement d'écran)
    
    <Tip>Alter needs these permissions to provide full functionality like reading selected text and capturing context.</Tip>
  </Accordion>
</AccordionGroup>

### Étape 2 : configuration initiale

<AccordionGroup>
  <Accordion icon="user" title="Configurez votre profil">
    Ouvrez **Paramètres** (⌘,) et accédez à l'onglet **Profil** :
    
    - Parlez à Alter de vous et de votre rôle
    - Partagez ce sur quoi vous travaillez
    - Définissez votre style de communication préféré
    - Ajoutez des références techniques ou des explicatifs pour les Tools que vous utilisez
    
    Cela aide Alter à fournir des réponses plus pertinentes et personnalisées.
  </Accordion>
  <Accordion icon="keyboard" title="Configurez vos raccourcis clavier">
    Dans **Paramètres > Valeurs par défaut** :
    
    - Définir votre raccourci **Quick Menu** (par défaut : Alt+Espace)
    - Activez le **bouton de sélection** pour un accès rapide lors de la sélection de texte
    - Ajustez les autres préférences si nécessaire
  </Accordion>
</AccordionGroup>

### Étape 3 : Commencez à utiliser Alter

<Accordion icon="rocket" title="Essayez votre première interaction">
  1. **Ouvrir Alter** : passez la souris sur l'encoche de votre Mac ou appuyez sur le raccourci de votre menu rapide
  2. **Tapez une invite** : posez une question ou donnez une commande
  3. **Ajouter un contexte** : faites glisser les fichiers ou utilisez `@` pour ajouter du contexte
  4. **Utiliser la voix** : maintenez l'icône du micro ou appuyez sur la touche de dictée pour parler
  
  <Tip>Press `/` in the prompt box to see available AI models and select your favorite!</Tip>
</Accordion>
