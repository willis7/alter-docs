Utilisez ce guide pour ajouter Alter comme fournisseur personnalisé dans TypingMind.

## Ouvrir la configuration du modèle personnalisé

Dans TypingMind, allez à :

`Settings` -> `Models` -> `+ Add Custom Model`

## 1) Configuration de base

Définissez les champs comme suit :

- `Name` : `Alter`
- `API Type` : `OpenAI Chat Completions API`
- `Endpoint URL` : `https://alterhq.com/api/v1/chat/completions`
- `Model ID` : choisissez un ID parmi ceux pris en charge ci-dessous
- `Context Length` : valeur par défaut ou celle que vous préférez

![Configuration de base TypingMind](/images/integrations/typingmind-basic-config.png)

## 2) Authentification

Dans la section Authentification :

- `Authentication Type` : `No authentication`

Remarque : nous transmettons la clé API manuellement via Custom Headers à l'étape suivante.

## 3) Configuration avancée (obligatoire)

Dépliez `Advanced` en bas, puis ajoutez une paire Custom Header :

- `Key` (gauche) : `Authorization`
- `Value` (droite) : `Bearer <YOUR_API_KEY>`

Remplacez `<YOUR_API_KEY>` par votre vraie clé Alter API.
Vérifiez qu'il y a bien un espace entre `Bearer` et votre clé.

![En-tête Autorisation TypingMind](/images/integrations/typingmind-auth-header.png)
