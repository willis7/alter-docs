---
title: "同梱の検索ツール"
description: "外部アカウント不要でAlter同梱のWeb検索ツールを使う"
---

Alterには2つのWeb検索ツールが同梱されています。

これらは、ネイティブ Web 検索を持たないモデル（ローカルモデルを含む）を拡張する時に有効です。

<Info>
  Tavily、LinkUp、Firecrawl、Exa と違い、これらは Alter の利用に含まれており、外部プロバイダー アカウントは不要です。
</Info>

## 比較

<CardGroup cols={2}>
  <Card title="Web Search (DuckDuckGo)" icon="compass">
    API キーは不要です。無料ですぐに利用可能です。スニペットベースなので高速ですが詳細は省略します。
  </Card>
  <Card title="Web Search Gemini" icon="sparkles">
    Google 検索 + Gemini 2.5 Flash。より詳細でソース付き。Alter Pro または Lifetime が必要です。
  </Card>
</CardGroup>

## 使い分け

- 無料アカウントや高速確認には **Web 検索 (DuckDuckGo)**。
- Pro/Lifetime での深い要約とソースが必要なら **Web Search Gemini**。

## 有効化

1. **ツール マネージャー** を開きます。
2. **ローカル ツール** に移動します。
3. **Web 検索** または **Web 検索 Gemini** を有効化します。

<Tip>
  2 つを同時に有効化しないでください。タスクごとに 1 つを有効化するか、使うツールを理解してください。
</Tip>
