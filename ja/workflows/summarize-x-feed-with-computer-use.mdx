title: "コンピューターの使用で X フィードを要約する"
description: "Alter Computer Use を使用して、X タイムラインに表示されている内容を読み取り、構造化された概要を生成します"
---

API アクセスが制限されていても、X (Twitter) フィードからの迅速な分析が必要な場合は、このワークフローを使用します。

## これが役立つこと

フィードを、Alter が画面から読み取るライブ ソースのように扱うことができます。 API からデータを取得する代わりに、Alter はブラウザーに現在表示されている内容を要約し、ダイジェストを繰り返し作成するのに役立ちます。

## で動作します

- アプリ: ブラウザの X
- ツールの種類: コンピューターの使用 (画面ベース)
- 権限: アクセシビリティと画面録画

## オルタが今できること

Alter は、表示されているフィード コンテンツを読み取り、画面上の投稿を要約し、繰り返しトピックを抽出し、画面上の複数のバッチを 1 つのダイジェストにマージできます。

## Alter がまだ実行していないこと

- このワークフローには公式の X API は使用されません。
- 現在表示されていない投稿については、スクロールして再度質問しない限り、要約することはできません。
- これは、完全な X アカウント自動化フロー (投稿、モデレーション、または分析パイプライン) ではありません。

## 現実的なワークフロー

<Steps>
  <Step title="Capture batch 1">
    タイムラインを開き、Alter に表示されている投稿を要約してもらいます。
  </Step>
  <Step title="Capture batch 2">
    下にスクロールして、新しい投稿に焦点を当てた 2 番目の概要を求めます。
  </Step>
  <Step title="Merge into digest">
    Alter に両方の概要を 1 つの簡潔なレポートに結合するよう依頼します。
  </Step>
</Steps>

## プロンプトのコピー/貼り付け

- 「X フィードに現在表示されているツイートを要約します。」
- 「表示されている投稿の上位 5 つのトピックをリストし、それぞれについて 1 行の証拠を付けます。」
- 「下にスクロールして次のバッチをキャプチャし、新しいトピックのみを要約します。」
- 「最後の 2 つのフィード概要を 1 つの簡潔な日次ダイジェストに結合します。」

## 有効にする方法

<Steps>
  <Step title="Enable Computer Use tools">
    Alter を開く -> 3 つの点をクリック -> **ツール マネージャー** -> **ローカル ツール** -> **Alter** ツール (コンピューター使用) を有効にします。
  </Step>
  <Step title="Grant required macOS permissions">
    Alter 設定で **アクセシビリティ** と **画面録画** を許可し、Alter がアプリ ウィンドウを読み取って操作できるようにします。
  </Step>
  <Step title="Run a quick test on X">
    X タイムラインを開いて、「画面に表示されている投稿を要約してください」と尋ねます。
  </Step>
</Steps>

## 期待されること

- 最高の品質は、短い反復ループ (要約 -> スクロール -> 要約 -> マージ) から得られます。
- 間違ったウィンドウがターゲットになっている場合は、ブラウザ ウィンドウを前面に持ってくるか、Alter にウィンドウをリストして正しいウィンドウをターゲットにするように依頼します。
