このガイドでは、Msty.ai を Alter API Router に接続します。

## provider 設定を開く

Msty.ai で、次を設定する provider/model 設定画面を開きます:

- API endpoint（base URL）
- API key
- モデル選択

## 1) API key

API key フィールドに Alter key を貼り付けます:

`sk-...`

Alter の `Settings` -> `Router` -> `Alter API Keys` でキーを生成します。

## 2) 推論 endpoint

endpoint を次の値に設定します:

```text
https://alterhq.com/api/v1
```

## 3) モデルを取得して選択

endpoint + API key 設定後、Msty.ai で **Fetch models** をクリックします。

Msty.ai が利用可能な Alter モデルを読み込み、一覧から選択できます。
モデル ID を手動入力する必要はありません。

![Msty.ai endpoint と API key の設定](/images/integrations/msty-endpoint-api-key.jpeg)

## 接続確認

1. Msty.ai で設定を保存
2. 簡単な prompt を送信
3. 失敗した場合は endpoint と API key を確認し、**Fetch models** を再実行
