## 連携チェックリスト

1. base URL を `https://alterhq.com/api` に設定する（必要なら `/v1`）
2. 認証 header を `Authorization: Bearer YOUR_API_KEY` に設定する
3. モデル ID は `Provider#Model-name` 形式を使う
4. 初回リクエスト前にモデル一覧を取得する
