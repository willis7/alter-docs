このガイドでは、TypingMind に Alter をカスタム provider として追加します。

## カスタムモデル設定を開く

TypingMind で次に進みます:

`Settings` -> `Models` -> `+ Add Custom Model`

## 1) 基本設定

次のように設定します:

- `Name`: `Alter`
- `API Type`: `OpenAI Chat Completions API`
- `Endpoint URL`: `https://alterhq.com/api/v1/chat/completions`
- `Model ID`: 下の対応 ID から選択
- `Context Length`: 既定値または任意の値

![TypingMind 基本設定](/images/integrations/typingmind-basic-config.png)

## 2) 認証

Authentication セクションで:

- `Authentication Type`: `No authentication`

注: API key は次の手順で Custom Headers から手動で渡します。

## 3) 高度な設定（必須）

下部の `Advanced` を開き、Custom Header のペアを追加します:

- `Key`（左）: `Authorization`
- `Value`（右）: `Bearer <YOUR_API_KEY>`

`<YOUR_API_KEY>` を実際の Alter API key に置き換えてください。
`Bearer` とキーの間に半角スペースを 1 つ入れてください。

![TypingMind Authorization Header](/images/integrations/typingmind-auth-header.png)
