URL スキームを使用して Alter を他のアプリと統合するには、このガイドを使用してください。

## ビデオチュートリアル

- https://www.youtube.com/watch?v=X-XJiveQ-Wc

## 導入

Alter は x-callback-url をサポートしているため、アプリは相互にアクションをトリガーできます。
あなたはできる：

- 外部アプリから Alter アクションをトリガーする
- Alter から外部アプリのアクションをトリガーする
- 入力データを URL で渡す
- 元のアプリに戻ります (アプリのサポートに応じて)

## 仕組み

1. `alter://...` URL を介して、別のアプリから Alter アクションをトリガーします。
2. アプリの URL スキームを使用して、Alter から外部アプリのアクションをトリガーします。

## 外部アプリから Alter アクションをトリガーする

1. アクションエディタを開きます。
2. アクションを選択します。
3. `General` で、**URL Callback** セクションから URL をコピーします。
4. URL を別のアプリ (ブラウザ、メモ、ショートカットなど) に貼り付けます。

![Alter バック コール URL](/images/integrations/alter-callback-urls-focus.webp)

## Alter から外部アプリ アクションをトリガーする

1. ターゲット アプリ スキームを使用して、Alter でクリック可能な URL を作成します。
2. そのアプリで必要とされるとおりにパラメータを正確にフォーマットします。
3. クリックして実行し、アクションの動作を確認します。

## 例

### Alter コールバックの例

- 検索クエリ:
  - __コード_0__
  - <a href="alter://action/ask-web?input=What+is+Alter+MacOS">Alter で開く</a>

- インプットのないビジネスストラテジスト:
  - __コード_0__
  - <a href="alter://action/business-strategist-gpt">Alter で開く</a>

- ビジネス ストラテジストからの意見:
  - __コード_0__
  - <a href="alter://action/business-strategist-gpt?input=Explain+Red+Ocean+Strategy">Alter で開く</a>

### クロスアプリの例

## テストとトラブルシューティング

- URL スキームとパラメータを確認します。
- 特殊文字を URL エンコードします。
- ターゲット アプリがアクションをサポートしていることを確認します。
- アプリのコールバック ドキュメント URL を Alter に貼り付けて、構文/ルールを確立します。
