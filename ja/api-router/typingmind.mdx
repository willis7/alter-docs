このガイドでは、TypingMind に Alter をカスタムプロバイダーとして追加します。

## カスタムモデル設定を開く

TypingMindで次に行きます:

`Settings` -> `Models` -> `+ Add Custom Model`

## 1) 基本設定

次のように設定します:

- `Name`: `Alter`
- `API Type`: `OpenAI Chat Completions API`
- `Endpoint URL`: `https://alterhq.com/api/v1/chat/completions`
- `Model ID`: 対応下のIDから選択
- `Context Length`:デフォルト値または任意の値

![TypingMind基本設定](/images/integrations/typingmind-basic-config.png)

## 2) 認証

認証セクションで:

- `Authentication Type`: `No authentication`

注: API キーは次の手順で Custom Headers から手動で渡します。

## 3) 高度な設定（必須）

下部の `Advanced` を開き、カスタムヘッダーのペアを追加します:

- `Key`（左）: `Authorization`
- `Value`（右）: `Bearer <YOUR_API_KEY>`

`<YOUR_API_KEY>` を実際の Alter API キーに置き換えてください。
`Bearer` とキーの間半角スペースを1つ入れてください。

![TypingMind 認証ヘッダー](/images/integrations/typingmind-auth-header.png)
