</Info>

## 対話する 2 つの方法

<CardGroup cols={2}>
  <Card title="ハブ" icon="layout">
    タブ、サイドバー、含むような管理を備えた完全なワークスペース
  </Card>
  <Card title="クイックパネル" icon="message-square">
    軽量のフローティングパネルで素早い書き込みや音声コマンドを実行可能
  </Card>
</CardGroup>

---

### 一般的な概念

Alter は、AI と対話するための 2 つの主要なインターフェイスを提供します。

<AccordionGroup>
  <Accordion icon="layout-dashboard" title="ハブ">
    タブ、サイドバー、およびこれらを含むワークスペース管理を備えたメイン ウィンドウ インターフェイス。以下に最適:
    - 長時間にわたる会話
    - ワークスペース管理
    - 複数のコンテキストによるマルチタスク
    - 会議記録の確認
  </Accordion>
  
<Accordion icon="message-circle" title="クイックパネル">
    ノッチまたはホットキーからアクセスできる軽量フローティング パネル。以下に最適:
    - クイック書き込み
    - 音声コマンド
    - コンテキストの高速な追加
    - 迅速なインタラクション
  </Accordion>
</AccordionGroup>

## ハブインターフェース

<Frame>
  ハブは 3 つの主要なセクションに分かれています。
  
1. **サイドバー** - 会話、ワークスペース、トランスクリプトなどのナビゲーション
  2. **メイン コンテンツ エリア** - アクティブなコンテンツを表示するタブ付きインターフェイス
  3. **入力** - プロンプトエリアボックスとコンテキストの管理
</Frame>

### ハブを開く

<Steps>
  <Step title="キーボードショートカット">
    `Shift + Cmd + H`を押してください
  </Step>
  <Step title="メニューバー">
    Alter メニュー バーアイコンから [ハブを開く] を選択します。
  </Step>
  <Step title="会話から">
    フローティングパネルから会話リンクを開く
  </Step>
</Steps>

### タブの種類

<CardGroup cols={3}>
  <Card title="会話" icon="message-square">
    AIモデルとのチャットセッション
  </Card>
  <Card title="__期間_6__" icon="folder-open">
    ファイル、解決、アプリのコレクション
  </Card>
  <Card title="成績証明書" icon="file-audio">
    会議の文字起こしと録音
  </Card>
  <Card title="カレンダーイベント" icon="calendar">
    予定されている会議やイベント
  </Card>
  <Card title="スケジュールされたジョブ" icon="clock">
    自動化されたタスクとワークフロー
  </Card>
</CardGroup>

### キーボードショートカット

|ショートカット |アクション |
|----------|----------|
| __コード_0__ |サイドバーの表示/非表示を切り替える |
| __コード_0__ |アクティブなタブを閉じる |
| __コード_0__ |次のタブ |
| __コード_0__ |前のタブ |
| __コード_0__ |すべてのコンテキストをクリア |
| __コード_0__ |アクティブな会話を削除 |

### コンテキストの管理

<Steps>
  <Step title="コンテキストを追加する">
    ファイルをアンドドロップするか、プロンプトボックスで`@`を使用します
  </Step>
  <Step title="ナビゲート">
    コンテキスト間を移動するには矢印キーを使用します
  </Step>
  <Step title="クリア">
    すべてのコンテキストを削除するには、`Cmd + K` を押してください
  </Step>
</Steps>

## クイックパネルインターフェース

クイック パネル (ノッチからアクセス可能) は 3 つのセクションに分かれています。

<CardGroup cols={3}>
  <Card title="1. コンテキスト" icon="layers">
    選択したファイル、アプリ、考え方
  </Card>
  <Card title="2. プロンプト" icon="edit">
    リクエストを入力または音声で入力する場所
  </Card>
  <Card title="3. 追加の選択" icon="plus">
    モデルとへアクションの素早いアクセス
  </Card>
</CardGroup>

### ナビゲーション

クイックパネルでキーボードショートカットを使用します。
- `←`、`↑`、`→`、`↓` - オブジェクト間を移動する
- `Enter` - オブジェクトを選択します
- `@` - コンテキストに要素を追加する
- `#` - 会話履歴を表示
- `>` - 会議履歴を表示
- `/` - モデルのリストと選択
- `?` - ヘルプメニューを表示する

<Note>
  キーボードショートカットの詳細については、以下の **プロンプト ボックス** セクションを参照してください。
</Note>

### キーボードナビゲーション

|キー |アクション |
|-----|----------|
| `←` `↑` `→` `↓` |コンテキストをナビゲートする |
| __コード_0__ |選択したものを追加/削除 |
| __コード_0__ |すべてのコンテキストを削除 |

### クリップボードの使用

コンテンツを直接貼り付けます: `Cmd + V`

クリップボードの内容は、Alter の変換テキストに自動的に追加されます。

## プロンプトボックス

**プロンプト ボックス** は魔法が起こる場所です。

<Frame>
  プロンプトまたはAlter アクション タイトルの入力を開始して、AI モデルに送信します。
</Frame>

### キーボードショートカット

| 異議 | アクション |
|----------|----------|
| __コード_0__ |アクティブなモデルを含むプロンプトを送信 |
| __コード_0__ |ウェブを検索 (Gemini経由) |
| __コード_0__ |変更 |
| __コード_0__ |要素を変換テキストに追加する |
| __コード_0__ |会話履歴を表示 |
| __コード_0__ |会議の文字起こし履歴を表示 |
| __コード_0__ |モデルをリストして選択 |
| __コード_0__ |ヘルプメニューを表示 |

<Tip>
  **の公開:** アクションのタイトルに任意の文字を入力してアクションをトリガーします。
</Tip>

## 会話ウィンドウ

### ウィンドウのサイズ変更

ワークフローに合わせて会話ウィンドウのサイズをカスタマイズします。

<CardGroup cols={2}>
  <Card title="幅" icon="move-horizontal">
    サイズを変更するには、エッジを水平にします
  </Card>
  <Card title="身長" icon="move-vertical">
    サイズを変更するには、端を垂直にします
  </Card>
  <Card title="レイアウト" icon="layout">
    他のアプリケーションと併用可能
  </Card>
  <Card title="持続的" icon="save">
    サイズはのセッションのために将来記憶されます
  </Card>
</CardGroup>

<Info>
  **用途:** 並行作業、長い文書のレビュー、画面共有、さまざまなディスプレイ設定
</Info>

### 連続的なスピーチからプロンプトまで

キーボードに触れずに長時間音声会話を行うことができます。

<Steps>
  <Step title="音声ホットキーを押し続ける">
    `Fn` (または設定したキー) を押し続けます。
  </Step>
  <Step title="自然に話す">
    プロンプトを大声で言ってください
  </Step>
  <Step title="リリース">
    放って会話に送信します
  </Step>
  <Step title="続けて">
    各音声プロンプトが同じスレッドに追加されます
  </Step>
</Steps>

<Tip>
  **ハンズフリーのワークフロー:** ブレーンストーミング、ディクテーション、または忙しいときに最適です。
</Tip>

## 会話履歴

### チャット履歴の管理

プロンプト ボックスに `#` と入力するか、ハブのサイドバーを使用して、履歴にアクセスします。

<Frame>
  ![タグ履歴付き](/images/other/history-tags.webp)
</Frame>

### タグで整理する

`#personal` のようなタグを追加して、チャットを分類します。

```
# #personal #food
```

複数のタグに一致する会話を検索します。

### 会話の名前を変更する

<Steps>
  <Step title="右クリック">
    リスト内の任意の会話を右クリックします
  </Step>
  <Step title="「名前の変更」を選択します">
    メニューから`Rename`を選択します
  </Step>
  <Step title="新しい名前を入力してください">
    わかりやすい名前を付けてください
  </Step>
</Steps>

### 会話のエクスポート

ドキュメント化または共有のためにマークダウンとしてエクスポートします。

<Steps>
  <Step title="会話を見つける">
    履歴から見つけてください
  </Step>
  <Step title="右クリック">
    コンテキストメニューを開く
  </Step>
  <Step title="エクスポートを選択します">
    エクスポートオプションを選択します
  </Step>
  <Step title="保存">
    場所を選択し、.mdファイルとして保存します
  </Step>
</Steps>

<Card title="輸出用途" icon="download">
  - ドキュメント
  - チームとの共有
  - 重要なディスカッションをアーカイブする
  - 他のツールでの使用
</Card>
