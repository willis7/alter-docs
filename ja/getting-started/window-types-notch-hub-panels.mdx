### 一般的な概念

Alter は、AI と対話するための 2 つの主要なインターフェイスを提供します。

<AccordionGroup>
  <Accordion icon="layout-dashboard" title="The Hub">
    タブ、サイドバー、および包括的なワークスペース管理を備えたメイン ウィンドウ インターフェイス。以下に最適:
    - 延長された会話
    - ワークスペース管理
    - 複数のコンテキストによるマルチタスク
    - 会議記録の確認
  </Accordion>
  
<Accordion icon="message-circle" title="Quick Panel">
    ノッチまたはホットキーからアクセスできる軽量フローティング パネル。以下に最適:
    - クイッククエリ
    - 音声コマンド
    - 高速コンテキストの追加
    - 迅速なインタラクション
  </Accordion>
</AccordionGroup>

## ハブインターフェース

<Frame>
  ハブは 3 つの主要なセクションに分かれています。
  
1. **サイドバー** - 会話、ワークスペース、トランスクリプトなどのナビゲーション
  2. **メインコンテンツエリア** - アクティブコンテンツを表示するタブ付きインターフェース
  3. **入力エリア** - プロンプト ボックスとコンテキストの管理
</Frame>

### ハブを開く

<Steps>
  <Step title="Keyboard shortcut">
    `Shift + Cmd + H`を押してください
  </Step>
  <Step title="Menu bar">
    メニューバーの変更アイコンから「ハブを開く」を選択します。
  </Step>
  <Step title="From a conversation">
    フローティング パネルから会話リンクを開く
  </Step>
</Steps>

### タブの種類

<CardGroup cols={3}>
  <Card title="Conversations" icon="message-square">
    AIモデルとのチャットセッション
  </Card>
  <Card title="Workspaces" icon="folder-open">
    ファイル、解決、アプリのコレクション
  </Card>
  <Card title="Transcripts" icon="file-audio">
    会議の文字起こしと録音
  </Card>
  <Card title="Calendar Events" icon="calendar">
    予定されている会議やイベント
  </Card>
  <Card title="Scheduled Jobs" icon="clock">
    自動化されたタスクとワークフロー
  </Card>
</CardGroup>

### キーボード ショートカット

|ショートカット |アクション |
|----------|----------|
| `Cmd + B` |サイドバーの表示/非表示を切り替える |
| `Cmd + W` |アクティブなタブを閉じる |
| `Cmd + Option + →` |次のタブ |
| `Cmd + Option + ←` |前のタブ |
| `Cmd + K` |すべてのコンテキストをクリア |
| `Cmd + Delete` |アクティブな会話を削除 |

### コンテキストの管理

<Steps>
  <Step title="Add context">
    ファイルをドラッグ アンド ドロップするか、プロンプト ボックスで `@` を使用します
  </Step>
  <Step title="Navigate">
    コンテキスト間を移動するには矢印キーを使用します
  </Step>
  <Step title="Clear">
    `Cmd + K` を押してすべてのコンテキストを削除します
  </Step>
</Steps>

### インスペクターパネル

サイドバーのアイコンまたはインスペクター ボタンを使用してインスペクターを切り替えます。

## クイックパネルインターフェース

クイック パネル (ノッチからアクセス可能) は 3 つのセクションに分かれています。

<CardGroup cols={3}>
  <Card title="1. Context" icon="layers">
    選択したファイル、アプリ、リファレンス
  </Card>
  <Card title="2. Prompt" icon="edit">
    リクエストを入力または音声で入力する場所
  </Card>
  <Card title="3. Additional Selection" icon="plus">
    モデルとアクションへの素早いアクセス
  </Card>
</CardGroup>

### ナビゲーション

クイック パネルでキーボード ショートカットを使用します。
- `←`、`↑`、`→`、`↓` - オブジェクト間を移動する
- `Enter` - オブジェクトを選択してください
- `@` - コンテキストに要素を追加します
- `#` - 会話履歴を表示
- `>` - 会議履歴を表示
- `/` - モデルをリストして選択します
- `?` - ヘルプ メニューを表示

<Note>
  キーボード ショートカットの詳細については、以下の **プロンプト ボックス** セクションを参照してください。
</Note>

### キーボードナビゲーション

|キー |アクション |
|-----|----------|
| `←` `↑` `→` `↓` |コンテキストをナビゲートする |
| `Space` |選択したものを追加/削除 |
| `Cmd + K` |すべてのコンテキストを削除 |

### クリップボードの使用

コンテンツを直接貼り付けます: `Cmd + V`

クリップボードの内容は、Alter の変換テキストに自動的に追加されます。

## プロンプトボックス

**プロンプト ボックス** は魔法が起こる場所です。

<Frame>
  プロンプトまたは変更アクションのタイトルの入力を開始して、AI モデルに送信します。
</Frame>

### キーボード ショートカット

| 異議 | アクション |
|----------|----------|
| `Enter` |アクティブなモデルを含むプロンプトを送信 |
| `Cmd + Enter` |ウェブを検索 (Gemini経由) |
| `Shift + Enter` |改行 |
| `@` |要素をコンテキストに追加する |
| `#` |会話履歴を表示 |
| `>` |会議の議題履歴を表示 |
| `/` |モデルをリストして選択 |
| `?` |ヘルプメニューを表示 |

<Tip>
  **アクションの呼び出し:** アクションのタイトルに任意の文字を入力してアクションをトリガーします。
</Tip>

## 会話ウィンドウ

### ウィンドウのサイズ変更

ワークフローに合わせて会話ウィンドウのサイズをカスタマイズします。

<CardGroup cols={2}>
  <Card title="Width" icon="move-horizontal">
    サイズを変更するには、エッジを水平にドラッグします
  </Card>
  <Card title="Height" icon="move-vertical">
    サイズを変更するには、端を垂直にドラッグします
  </Card>
  <Card title="Layout" icon="layout">
    他のアプリケーションと併用可能
  </Card>
  <Card title="Persistent" icon="save">
    サイズは将来のセッションのために記憶されます
  </Card>
</CardGroup>

<Info>
  **用途:** 並列作業、長い文書のレビュー、画面共有、さまざまなディスプレイ設定
</Info>

### 連続的なスピーチからプロンプトまで

キーボードに触れずに長時間音声会話を行うことができます。

<Steps>
  <Step title="Hold voice hotkey">
    `Fn` (または設定したキー) を長押しします。
  </Step>
  <Step title="Speak naturally">
    プロンプトを大声で言ってください
  </Step>
  <Step title="Release">
    放して会話に送信します
  </Step>
  <Step title="Keep going">
    各音声プロンプトが同じスレッドに追加されます
  </Step>
</Steps>

<Tip>
  **ハンズフリーのワークフロー:** ブレーンストーミング、ディクテーション、または手が忙しいときに最適です。
</Tip>

## 会話履歴

### チャット履歴の管理

プロンプト ボックスに `#` と入力するか、ハブのサイドバーを使用して、履歴にアクセスします。

<Frame>
  ![タグ付き履歴](/images/other/history-tags.webp)
</Frame>

### タグによる整理

`#personal` のようなタグを追加して、チャットを分類します。

```
# #personal #food
```

複数のタグに一致する会話を検索します。

### 会話の名前を変更する

<Steps>
  <Step title="Right-click">
    リスト内の任意の会話を右クリックします
  </Step>
  <Step title="Select Rename">
    メニューから `Rename` を選択します
  </Step>
  <Step title="Type new name">
    わかりやすい名前を付けてください
  </Step>
</Steps>

### 会話のエクスポート

ドキュメント化または共有のためにマークダウンとしてエクスポートします。

<Steps>
  <Step title="Find conversation">
    履歴から見つけてください
  </Step>
  <Step title="Right-click">
    コンテキストメニューを開く
  </Step>
  <Step title="Select Export">
    エクスポートオプションを選択します
  </Step>
  <Step title="Save">
    場所を選択し、.mdファイルとして保存します
  </Step>
</Steps>

<Card title="Export Uses" icon="download">
  - ドキュメント
  - チームとの共有
  - 重要なディスカッションのアーカイブ
  - 他のツールでの使用
</Card>
