このページを使用して、タスクに重要なファイルと削除を使用して変更を実行します。

## Finder またはアプリからコンテキストを追加

ファイルまたは解決をノッチまたはハブに行うことで、コンテンツを追加できます。 **右クリック > [Alter に追加]** を使用することもできます。

## サポートされているコンテンツ

<CardGroup cols={2}>
  <Card title="Documents" icon="file-text">
    `.xlsx`、`.docx`、`.pdf`、`.txt`、`.md`、`.org`、`.rtf`
  </Card>
  <Card title="Images" icon="image">
    `.jpg`、`.png`、`.heic`、`.gif`、`.tiff`、`.svg`
  </Card>
  <Card title="Web content" icon="globe">
    `.html`、URL、記録されたテキスト
  </Card>
  <Card title="Email" icon="mail">
    `.eml` と Apple Mail メッセージ
  </Card>
  <Card title="Audio" icon="music">
    `.mp3`、`.wav`、`.m4a`、`.aac`、`.flac`
  </Card>
  <Card title="Code" icon="code">
    ソースファイル、設定ファイル、スクリプト
  </Card>
</CardGroup>

<Note>
  Alter は、ディレクトリのドロップ、画像と PDF の OCR、および自動テキスト エンシード検出をサポートしています。
</Note>

## 音声とビデオの文字起こし

オーディオまたはビデオを追加すると、Alter は自動的に文字起こしを開始します。

<Steps>
  <Step title="Drop one or more files">
    メディア ファイルをノッチまたはハブにドラッグします。
  </Step>
  <Step title="Wait for processing">
    文字起こしが自動的に始まります。
  </Step>
  <Step title="Ask targeted questions">
    「重要な決定事項を要約する」または「所有者ごとにアクション アイテムをリストする」を試してください。
  </Step>
</Steps>

<Warning>
  画像解析には視覚機能を備えたモデルを使用してください。
</Warning>
