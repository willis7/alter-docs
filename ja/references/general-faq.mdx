## Alter は他のツールと何が違うのですか?

Alter は、現在の作業状況を理解し、アプリと直接統合するネイティブ macOS AI アシスタントです。

## システム要件は何ですか?

Alter は macOS Ventura 13 以降をサポートし、Apple Silicon と Intel Mac の両方で実行します。

## どの macOS アプリが Appsense と互換性がありますか?

すべての macOS アプリは、すぐに使える Alter Appsense と互換性があります。

## Alter はどのように時間を節約しますか?

Alter は、ワークフローのコンテキストに応じたヘルプ (作成サポート、会議の記録と概要、ファイル コンテキストのドラッグ アンド ドロップなど) を提供します。

## Alter ではどのような AI モデルを使用できますか?

Alter は主要なプロバイダーをサポートし、新しくリリースされたモデルを迅速に追加します。

- ビジョン対応モデル (Gemini、Claude、GPT-5、Pixtral など)
- テキスト中心のモデル (Llama、Qwen、DeepSeek など)

## 自分の API キーを使用できますか?

はい。 [独自の API キーを使用する方法](/how-to/use-byok) を参照してください。

## Alter をオフラインまたはローカル モデルで使用できますか?

はい。 Alter は、ローカル処理用に Ollama および LM Studio をサポートします。 [プライバシーを重視したローカルモデルのワークフロー](/use-cases/local-model-privacy)を参照してください。

注: LM Studio のサポートはデバイス クラスによって異なる場合があります。 Ollama は Mac で広くサポートされています。

## Alter をルーターで使用できますか?

はい。 Alter は、設定にエンドポイント + API キーを入力することで、OpenAI 互換ルーターで動作します。
Alter は、API ゲートウェイを介したルーター自体でもあります。

## Alter は私のデータを参照できますか?

モデルに送信されるプロンプトとデータは、転送中に暗号化されます。
Alter は匿名の使用状況分析を収集します。これは設定で無効にすることができます。
