### グローバルナビゲーション

(ターゲット ウィンドウがすでにアクティブでない限り) 複数のウィンドウで使用できます。

- **Shift + Cmd + H**: Alter ハブを開く
- **Shift + Cmd + E**: アクション エディターを開く
- **Shift + Cmd + T**: Tools Manager を開きます
- **Alt + Space** (デフォルト): オープン ノッチ — **[設定] > [デフォルト] > [オープン ノッチ キーバインド]** で構成可能
- **Alter クイック メニュー ショートカット**: どこからでも Alter を開く — **[設定] > [一般] > [クイック アクセス]** で構成可能
- **音声入力ホットキー**: 音声入力の開始 — **設定 > ディクテーション > ショートカット** で構成可能 (デフォルト: Fn キー)

### クイックパネル

クイックパネルを開いた状態（切り欠き側）：

- **←**、**↑**、**→**、**↓**: オブジェクト間を移動します。
- **Enter**: オブジェクトを選択します

### コンテキストセクション

- **←**、**↑**、**→**、**↓**: コンテキスト間を移動します。
- **スペースバー**: 選択したコンテキストを追加/削除します
- **Cmd + K**: アクティブな会話からすべてのコンテキストを削除します
- **Cmd + V**: クリップボードの内容を貼り付けてコンテキストに追加します

### プロンプトボックス

- **Enter**: アクティブなモデルのプロンプトを表示します
- **Cmd + Enter**: Web 検索を使用したプロンプト (Gemini Web 経由)
- **Shift + Enter**: 改行
- **@**: 要素をコンテキストに追加します
- **#**: Alter の会話履歴を表示
- **>**: 会議の文字起こしの履歴を表示します
- **/**: モデルをリストして選択します
- **?**: ヘルプメニューを表示

### Alter ハブ ウィンドウ

- **Cmd + F**: フォーカス検索 (結果を移動するには矢印キー)
- **Cmd + B**: サイドバーの表示/非表示を切り替えます
- **Cmd + T**: 新しいタブ (新しいチャットを開始)
- **Cmd + J**: Workspaceの作成 (アクティブなコンテキストが必要)
- **Cmd + Opt + O**: Workspaceに切り替えます...
- **Cmd + K**: すべてのコンテキストをクリアします (アクティブなコンテキストが必要です)
- **Cmd + Delete**: 会話を削除します (アクティブな会話が必要です)
- **Cmd + W**: タブを閉じる (最後のタブの場合はウィンドウを閉じる)
- **Cmd + Opt + W**: すべてのタブを閉じます (タブが 1 つ以上必要です)
- **Cmd + Opt + ←**: 前のタブ (1 つ以上のタブが必要)
- **Cmd + Opt + →**: 次のタブ (1 つ以上のタブが必要)
- **Cmd + 1...9**: 特定のタブ インデックスに切り替えます

#### Workspace (選択したWorkspaceが必要)

デフォルトのショートカットはありません。これらは macOS システム設定から設定できます。

- Workspace設定を開く
- Workspaceの名前を変更する
- 権限を設定する
- サンドボックスの切り替え
- サンドボックスコンテナの開始/停止
- サンドボックスでターミナルを開く

### アクションエディタウィンドウ

- **Cmd + F**: フォーカス検索
- **Cmd + N**: 新しいアクション
- **Cmd + D**: アクションを複製します (選択したアクションが必要です)
- **Cmd + E**: エクスポート アクション (選択したアクションが必要です)
- **Cmd + O**: インポートアクション
- **Cmd + Delete**: アクションを削除します (選択したアクションが必要です)
- **Cmd + W**: ウィンドウを閉じる
- **Cmd + [**: 前のセクション
- **Cmd + ]**: 次のセクション

### Tools Manager ウィンドウ

- **Cmd + W**: ウィンドウを閉じる
- **Cmd + F**: フォーカス検索
- **「スキル」を選択した場合**: Cmd + O (スキルのインポート)、Cmd + R (リストの更新)
- **「リモート MCP サーバー」を選択した場合**: Cmd + N (サーバーの追加)、Cmd + R (リストの更新)
- **「クイック アクション」が選択されている場合**: Cmd + N (新しいクイック アクション)、Delete (クイック アクションを削除、選択が必要)

### ライブミーティングパネル

会議中のみ利用可能:

- **Cmd + Shift + ↓**: ライブキャプションのオン/オフを切り替えます
- **Shift + Cmd + →**: ライブメモ帳を切り替えます

### Alter 設定

- **Cmd + ,**: 設定を開く (Alter ウィンドウが表示されている場合)
