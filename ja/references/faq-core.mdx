最初にこれらの短い回答を使用してから、ハウツーまたはガイドのコンテンツを拡張してください。

範囲: 製品内の使用方法に関する簡単な質問のみ (Alter の開始、コンテキスト、設定、ディクテーション、BYOK セットアップ)。

## Alter を開くにはどうすればよいですか?

1. ノッチの上にマウスを移動します。
2. `Alt+Space` を押します。
3. `Fn` を押しながら話してください。

![Alterを開く](/images/getting-started/open-notch.gif)

詳細については、[はじめる](/guides/getting-started) を参照してください。

## 自分の API キーを追加するにはどうすればよいですか?

1. 設定 (`Cmd+,`) を開きます。
2. 「API キー」に移動します。
3. プロバイダーを選択し、キーを入力します。

![カスタムAPIキー](/images/models/custom-api-keys.webp)

詳細な手順については、[ハウツー: 独自の API キーを使用する](/how-to/use-byok) を参照してください。

## プロンプトにコンテキストを追加するにはどうすればよいですか?

- ファイルをノッチにドラッグ アンド ドロップします。
- ファイルを右クリックし、[Alter に追加] を選択します。
- `@` を使用して、プロンプトからコンテキストを追加します。

![サイクルアンドドロップ](/images/context/drag-and-drop.gif)

詳細については、[Alter ウィンドウ](/guides/alter-window#the-context-section) を参照してください。

## 設定をすばやく変更するにはどうすればよいですか?

- Alter を開き、`Cmd+,` を押します。
- または、三点メニュー -> [設定] を開きます。

![設定メニュー](/images/settings/settings-menu.png)

詳細については、[Alter 設定](/guides/alter-settings) を参照してください。

## ディクテーションを設定するにはどうすればよいですか?

[設定] -> [ディクテーション] を選択し、プロセッサ、言語、置換を選択します。

![ディクテーション](/images/settings/dictation.webp)

完全なセットアップ手順については、[ディクテーション ガイド](/guides/dictation) を参照してください。
