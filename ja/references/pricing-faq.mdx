ユーザーがプラン、請求、フェアユース、割引、またはデバイス制限について質問する場合は、このリファレンスを使用してください。

範囲: 試用期間やライセンス制限を含む、すべての金額/プラン ポリシーに関する質問。
製品の機能とセットアップ フローについては、`references/general-faq.md` と `references/faq-core.md` を使用します。

## 計画の概要

- **無料 ($0)**: 独自の API キーまたはローカル モデルを使用したすべてのコア Alter 機能。クレジットカードは必要ありません。
- **Local+ (アドオン)**: 高度なローカル音声モデル (Whisper Pro および Parakeet V3)、最大 3 台のデバイス。
- **Pro (サブスクリプション)**: 50 以上のクラウド モデル、ルーター モデル (最高/高速/軽量)、API ゲートウェイ、リモート ツール、優先サポートへの管理されたアクセス。
- **生涯 (1 回限り)**: 定期請求なしのサブスクリプション プランと同じアクセス レベル。

## 試用版と無料使用法

### Alter は本当に無料ですか?

はい。 Alter は、ローカル モデルまたは独自の API キーを使用する場合、コア機能とともに無料で使用できます。

### 7 日間のトライアル後はどうなりますか?

試用後、アプリは BYOK/ローカル使用で機能し続けます。有料プランではアップグレードのプロンプトが削除され、管理されたモデルへのアクセスが提供されます。

## 無制限かつ公正な使用

### 「無制限」とはどういう意味ですか?

有料プランには、フェアユース ポリシーの対象となる、Alter アプリでの無制限のリクエストが含まれます。

### フェアユースはどのように機能しますか?

- 厳格なメッセージ上限ではなく、月ごとの予算システムを使用します。
- 予算を超過した場合、リクエストはアプリ内で通知される低コストのモデルにルーティングされる場合があります。
- API ゲートウェイには 1 日あたりの制限 (1 日あたり 200 リクエスト) があり、その後は 10 分あたり 1 リクエストにスロットルされます。
- 頻繁に使用される自律コーディング エージェントのトラフィックは、ポリシーに基づいて禁止されています。

## ルーター

### Alter ライトとは何ですか?

ライト/フラッシュモデルを選択し、速度/コストを重視したルーター。

### Alter はどれが最適ですか?

品質と速度のために強力なモデル (Sonnet ファミリなど) を選択するバランスの取れたルーター。

### Alter 高速とは何ですか?

非常に高スループットのモデルを優先した速度優先のルーター。

## デバイスとライセンス

### Alter を複数のデバイスで使用できますか?

はい、すべてのプランで可能です。アクティブな使用は、1 人のユーザーが一度に 1 台のデバイスに制限されます。

### プランごとのデバイス制限

- **ローカル+**: 最大 3 台のデバイスをインストール可能
- **プロ/ライフタイム**: 無制限のインストール
- **すべてのプラン**: アクティブなデバイスは一度に 1 台。ライセンス共有なし

## 割引

### 学生・学術割引

利用可能。大学のメールアドレスから `hi@alterhq.com` に連絡してください。

### 非営利割引

利用可能。組織の詳細については `hi@alterhq.com` にお問い合わせください。

## 対応のためのサポート ガイダンス

- **無料のコアアプリ**と**管理された有料モデルへのアクセス**の違いを明確にしてください。
- ユーザーが「無制限」について質問した場合は、フェアユースと API ゲートウェイの制限について言及します。
- デバイスの共有を求められる場合は、シングルユーザー ライセンスの言語を含めます。
- ユーザーに請求/サポートを案内します: `hi@alterhq.com`。
