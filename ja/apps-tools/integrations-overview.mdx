## __期間_5__

Alter は、Mac アプリケーションと直接統合する **17 以上の強力なローカル ツール** を提供します。他の AI アシスタントとは異なり、Alter は実際に自然言語 Mac を制御します。

<CardGroup cols={2}>
  <Card title="完全無料" icon="gift">
    みんなのローカルツールは誰でも無料で利用できます
  </Card>
  <Card title="オフラインでも動作" icon="wifi-off">
    インターネットは不要 – 完全にプライベート
  </Card>
  <Card title="ネイティブコントロール" icon="command">
    アプリについてだけでなく、実際にアプリを制御します
  </Card>
  <Card title="プライバシー第一" icon="shield">
    データがMacから離れることはありません
  </Card>
</CardGroup>

### ローカルツールの使用方法

<Steps>
  <Step title="Tools Manager を開く">
    Alter → **Tools Manager** の 3 つのポイントをクリックします。
  </Step>
  <Step title="ツールを有効にする">
    **ローカル ツール** セクションで、ツール (カレンダー、マップなど) を検索して有効にします。
  </Step>
  <Step title="自然言語を使用する">
    Alter にアクションの実行を依頼します。
    - 「ジョンに遅れそうなメッセージを送って」
    - 「明日の午後2時にカレンダーに会議を追加します」
    - 「未読メールを見せて」
  </Step>
</Steps>

<Note>
  Tools Manager のグローバルスイッチを使用してすべてのツールのオン/オフを切り替えます ([設定] > ツール)。
</Note>

## x-callback-url の統合

Alter は、アプリ間通信用の [x-callback-url](https://x-callback-url.com/) プロトコルをサポートします。

### x-callback-url とは何ですか?

特別な URL アクションを使用してアプリが相互にアクションをトリガーできるように標準。データを渡し、実行し、先に戻ります。

### 他のアプリからAlterをトリガーする

<Steps>
  <Step title="URLを取得する">
    エディターで、アクションを選択→「URLコールバック」セクション
  </Step>
  <Step title="スキームのコピー">
    そのアクションの `alter://` URL をコピーします
  </Step>
  <Step title="どこでも使用可能">
    ショートカット、メモ、ブラウザなどに貼り付けます。
  </Step>
</Steps>

**例：**
```
alter://action/ask-web?input=What+is+Alter+MacOS
```

### Alter から他のアプリをトリガーする

対象アプリのURLスキームを使用してクリック可能なリンクを作成します。

|アプリ | URLの例 | アクション |
|-----|------|----------|
|クマ | __コード_0__ |メモを作成する |
|もの| __コード_0__ |タスクを追加 |
| 発生 | __コード_0__ |ドラフトの作成 |

<Card title="サポートされているアプリ" icon="check-circle">
  Bear、Things、Drafts、OmniFocus、1Writer、Agenda、その他数百もの
</Card>

<Info>
  完全なドキュメントについては、[URL コールバック ガイド](/workflows/url-callbacks) を参照してください。
</Info>
