---
title: "実行する"
description: "UI 要素に対してクリック、塗りつぶし、ショートカット アクションを実行する"
---

Get Active App Context でターゲットを特定した後、このツールを使用して具体的な UI インタラクションを実行します。

## サポートされているアクション

- `click`: 対象の要素をクリックします。
- `fill`: テキストフィールドまたはテキストエリアにテキストを入力します。
- `shortcut`: キーボード ショートカットを送信します (例: `cmd+c`、`ctrl+shift+t`)。

## 入力

- `elementID` (`click` および `fill` に必須): アクティブなアプリ コンテキストの取得出力からの要素 ID。
- `text` (`fill` には必須): ターゲット入力に挿入する値。
- `shortcut` (`shortcut` には必須): ショートカット文字列。
- `windowID` (オプション): アクティブなウィンドウではなく、特定のウィンドウ (`pid/title`) をターゲットにします。

## 注意事項

- 要素 ID は、UI の変更やセッションのタイミングによって期限切れになります。要素が有効でなくなった場合は、コンテキストを再読み取りします。
- バックグラウンド ウィンドウ ターゲティングの場合、コンテキスト読み取りと実行ステップ全体で同じ `windowID` を使用します。
