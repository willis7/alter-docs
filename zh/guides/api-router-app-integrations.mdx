## 集成检查清单

1. 将 base URL 设为 `https://alterhq.com/api`（如有需要使用 `/v1`）
2. 将认证 header 设为 `Authorization: Bearer YOUR_API_KEY`
3. 使用 `Provider#Model-name` 格式的模型 ID
4. 首次请求前先调用一次模型列表
