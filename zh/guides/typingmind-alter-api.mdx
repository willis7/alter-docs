---
title: "在 TypingMind 中设置 Alter API"
description: "使用 OpenAI Chat Completions 格式将 TypingMind 连接到 Alter API Router"
---

使用本指南将 Alter 添加为 TypingMind 的自定义 provider。

## 打开自定义模型设置

在 TypingMind 中，前往：

`Settings` -> `Models` -> `+ Add Custom Model`

## 1）基础配置

按如下设置字段：

- `Name`: `Alter`
- `API Type`: `OpenAI Chat Completions API`
- `Endpoint URL`: `https://alterhq.com/api/v1/chat/completions`
- `Model ID`: 从下方支持的 ID 中选择
- `Context Length`: 默认值或你偏好的值

![TypingMind 基础配置](/images/integrations/typingmind-basic-config.png)

## 2）认证

在 Authentication 区域：

- `Authentication Type`: `No authentication`

说明：我们会在下一步通过 Custom Headers 手动传入 API key。

## 3）高级配置（必需）

展开底部 `Advanced`，然后添加一组 Custom Header：

- `Key`（左侧）：`Authorization`
- `Value`（右侧）：`Bearer <YOUR_API_KEY>`

将 `<YOUR_API_KEY>` 替换为真实的 Alter API key。
请确保 `Bearer` 和 key 之间有一个空格。

![TypingMind 授权 Header](/images/integrations/typingmind-auth-header.png)
