## 简介

Alter 支持 x-callback-url，因此应用程序可以相互触发操作。
您可以：

- 从外部应用程序触发 Alter 操作
- 从 Alter 触发外部应用程序操作
- 在 URL 中传递输入数据
- 返回原始应用程序（取决于应用程序支持）

## 它是如何工作的

1. 通过 `alter://...` URL 从另一个应用程序触发 Alter 操作。
2. 使用该应用程序的 URL 方案从 Alter 触发外部应用程序操作。

## 从外部应用程序触发 Alter 操作

1. 打开动作编辑器。
2. 选择一个操作。
3. 在 `General` 中，复制 **URL Callback** 部分中的 URL。
4. 将 URL 粘贴到另一个应用程序（浏览器、注释、快捷方式等）中。

！[Alter 回调 URL](/images/integrations/alter-callback-urls-focus.webp)

## 从 Alter 触发外部应用程序操作

1. 使用目标应用方案在 Alter 中创建可点击的 URL。
2. 完全按照该应用程序的要求设置参数格式。
3. 单击以运行并验证操作行为。

## 示例

### Alter 回调示例

- 搜索查询：
  - `alter://action/ask-web?input=What+is+Alter+MacOS`
  - <a href="alter://action/ask-web?input=What+is+Alter+MacOS">在 Alter 中打开</a>

- 没有输入的商业策略师：
  - `alter://action/business-strategist-gpt`
  - <a href="alter://action/business-strategist-gpt">在 Alter 中打开</a>

- 商业策略师的意见：
  - `alter://action/business-strategist-gpt?input=Explain+Red+Ocean+Strategy`
  - <a href="alter://action/business-strategist-gpt?input=Explain+Red+Ocean+Strategy">在 Alter 中打开</a>

### 跨应用示例

- 熊笔记创建：
  - `bear://x-callback-url/create?title=Demo%20Note`

## 测试和故障排除

- 验证 URL 方案和参数。
- URL 对特殊字符进行编码。
- 确认目标应用程序支持该操作。
- 将应用程序的回调文档 URL 粘贴到 Alter 中以遵循语法/规则。
